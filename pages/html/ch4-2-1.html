Теги <table>, <tr>, <td>, <th>, атрибуты colspan, rowspan.

Таблицы в HTML — это мощный инструмент для представления структурированной информации. Основные теги для создания таблиц — это <table>, <tr>, <td>, <th>, а также атрибуты colspan и rowspan, которые помогают объединять ячейки таблицы. Давай разберём их подробнее.

1. Тег <table> — создание таблицы

Тег <table> используется для создания таблицы. Он является контейнером для всех строк и ячеек таблицы.

Пример:

  
    <table>
      <!-- Внутри этого тега находятся строки и ячейки -->
    </table>

Особенности:

  • Тег <table> сам по себе не отображает данные. Он служит контейнером для других элементов, таких как строки и ячейки.
  • Для создания более сложных таблиц внутри можно использовать теги <thead>, <tbody>, и <tfoot> для логической группировки данных (хотя они не обязательны).

2. Тег <tr> — создание строки

Тег <tr> обозначает строку в таблице. Каждая строка таблицы состоит из одной или нескольких ячеек.

Пример:

  
    <table>
      <tr>
        <td>Ячейка 1</td>
        <td>Ячейка 2</td>
      </tr>
      <tr>
        <td>Ячейка 3</td>
        <td>Ячейка 4</td>
      </tr>
    </table>

Результат:

Ячейка 1 Ячейка 2
Ячейка 3 Ячейка 4

Особенности:

  • Все ячейки в строке должны находиться внутри тега <tr>.
  • Таблица может содержать любое количество строк, каждая из которых создаётся с помощью одного <tr>.

3. Тег <td> — создание ячейки

Тег <td> используется для создания ячейки в строке таблицы. Это стандартный тег для хранения данных.

Пример:

  
    <table>
      <tr>
        <td>Данные 1</td>
        <td>Данные 2</td>
      </tr>
    </table>

Результат:

Данные 1 Данные 2

Особенности:

  • <td> (table data) — это стандартная ячейка таблицы, используемая для размещения текстовой информации, изображений, ссылок и других данных.
  • Каждая строка (<tr>) может содержать любое количество ячеек <td>, которые будут отображаться в виде столбцов.

4. Тег <th> — создание заголовка таблицы

Тег <th> обозначает заголовок ячейки. Обычно используется для создания заголовков столбцов или строк. По умолчанию текст в этих ячейках отображается жирным и выравнивается по центру.

Пример:

  
    <table>
      <tr>
        <th>Имя</th>
        <th>Возраст</th>
      </tr>
      <tr>
        <td>Иван</td>
        <td>25</td>
      </tr>
    </table>

Результат:

Имя Возраст
Иван 25

Особенности:

  • <th> (table header) обозначает ячейку заголовка, обычно это первый ряд или первая колонка таблицы.
  • Заголовочные ячейки <th> можно использовать как для столбцов, так и для строк.
  • Для улучшения доступности можно использовать атрибуты scope="col" или scope="row", чтобы обозначить, что заголовок относится к конкретной колонке или строке:
  
    <th scope="col">Имя</th>
    <th scope="row">Иван</th>

5. Атрибут colspan — объединение колонок

Атрибут colspan используется для объединения нескольких ячеек по горизонтали (объединение колонок). Это полезно, когда нужно создать ячейку, которая должна охватывать более одного столбца.

Пример:

  
    <table>
      <tr>
        <th>Имя</th>
        <th>Возраст</th>
      </tr>
      <tr>
        <td colspan="2">Иван</td>
      </tr>
    </table>

Результат:

Имя Возраст
Иван

Здесь ячейка с именем "Иван" занимает два столбца.

Особенности:

  • Значение атрибута colspan указывает, сколько столбцов объединяет ячейка.
  • Если не указать этот атрибут, по умолчанию каждая ячейка занимает только один столбец.

6. Атрибут rowspan — объединение строк

Атрибут rowspan позволяет объединять ячейки по вертикали (объединение строк). Это полезно, когда нужно, чтобы одна ячейка охватывала несколько строк.

Пример:

  
    <table>
      <tr>
        <th>Имя</th>
        <th>Возраст</th>
      </tr>
      <tr>
        <td rowspan="2">Иван</td>
        <td>25</td>
      </tr>
      <tr>
        <td>26</td>
      </tr>
    </table>

Результат:

Имя Возраст
Иван 25
26

В этом примере ячейка с именем "Иван" занимает два ряда.

Особенности:

  • Значение атрибута rowspan указывает, сколько строк объединяет ячейка.
  • Как и с colspan, если не указывать атрибут, ячейка будет занимать только одну строку.

7. Пример сложной таблицы с colspan и rowspan

Вот пример таблицы, в которой используются оба атрибута:

  
    <table border="1">
      <tr>
        <th>Фамилия</th>
        <th>Имя</th>
        <th colspan="2">Контактные данные</th>
      </tr>
      <tr>
        <td rowspan="2">Иванов</td>
        <td>Иван</td>
        <td>Email</td>
        <td>ivan@example.com</td>
      </tr>
      <tr>
        <td>Петр</td>
        <td>Телефон</td>
        <td>+7 999 123 45 67</td>
      </tr>
    </table>

Результат:

Фамилия Имя Контактные данные
Иванов Иван Email ivan@example.com
Петр Телефон +7 999 123 45 67

В этом примере:

  • Первая строка объединяет две колонки для отображения заголовка "Контактные данные" с помощью атрибута colspan.
  • Вторая и третья строки объединяют ячейку с фамилией "Иванов" на два ряда с помощью атрибута rowspan.

8. Стилизация таблиц

Чтобы сделать таблицы более привлекательными и удобными для чтения, можно использовать CSS. Вот несколько примеров стилей для таблиц:

Пример с границами:

   
    <style>
      table {
        width: 100%;
        border-collapse: collapse;
        margin-top: 20px;
      }
      table, th, td {
        border: 1px solid #ddd;
      }
      th, td {
        padding: 10px;
        text-align: left;
      }
      th {
        background-color: #f2f2f2;
      }
    </style>
    
    <table>
      <tr>
        <th>Имя</th>
        <th>Возраст</th>
      </tr>
      <tr>
        <td>Иван</td>
        <td>25</td>
      </tr>
    </table>

Результат:

Имя Возраст
Иван 25

Здесь:

  • Границы таблицы, заголовков и ячеек добавляются с помощью свойства border.
  • Свойство border-collapse: collapse; убирает двойные границы между ячейками, делая таблицу более аккуратной.

9. Заключение

Использование таблиц в HTML — это мощный способ представления данных в структурированном виде. Важно правильно применять теги и атрибуты, такие как <table>, <tr>, <td>, <th>, а также colspan и rowspan для объединения ячеек. При этом стоит помнить о стилизации таблиц для улучшения их визуального восприятия.

Эти основы помогут тебе создавать простые и сложные таблицы для различных целей, будь то отображение статистики, расписаний или любой другой информации.
