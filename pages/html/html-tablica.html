Таблицы в HTML

Пример таблицы

Person 1 Person 2 Person 3
Emil Tobias Linus
16 14 10 
<!DOCTYPE html>
<html>
<style>
table, th, td {
  border:1px solid black;
}

th, td {
  background-color: #96D4D4;
}
</style>
<body>
                    
<table style="width:100%">
  <tr>
    <th>Person 1</th>
    <th>Person 2</th>
    <th>Person 3</th>
  </tr>
  <tr>
    <td>Emil</td>
    <td>Tobias</td>
    <td>Linus</td>
  </tr>
  <tr>
    <td>16</td>
    <td>14</td>
    <td>10</td>
  </tr>
</table>
                    
</body>
</html>

<table> - тег для таблицы

<tr></tr> - строка таблицы (table row)

<th></th> - ячейки заголовка таблицы (table header)

<td></td> - ячейки таблицы (table data)
