Теги <html>, <head>, <body>, структура HTML-документа.

  1. Тег <html>

    2. Тег <html> — это корневой элемент любого HTML-документа. Он охватывает всё содержимое страницы, начиная с заголовочной информации и заканчивая телом (контентом). Браузеры понимают, что всё внутри этого тега — это часть HTML-документа.

      
    <html lang="ru">
      <!-- Всё содержимое страницы -->
    </html>

    Атрибуты:

    • lang — указывает язык содержимого. Это полезно для поисковых систем и программ чтения с экрана. Например, lang="ru" обозначает, что страница на русском языке.
  2. Тег <head>

    3. Тег <head> содержит метаданные (данные о данных), которые не отображаются непосредственно на странице, но они важны для работы браузера, поисковых систем и других сервисов.

    Внутри <head> обычно находятся следующие элементы:

    • <meta> теги — для указания кодировки, описания страницы и ключевых слов для SEO.
    • <title> — задаёт заголовок страницы, который отображается на вкладке браузера.
    • Ссылки на внешние стили с помощью тега <link> (например, подключение CSS).
    • Скрипты можно также подключать в <head>, но часто их размещают перед закрывающим тегом <body>, чтобы не замедлять загрузку контента.

    Пример содержимого <head>:

      
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <meta name="description" content="Описание страницы для поисковиков">
      <title>Заголовок страницы</title>
      <link rel="stylesheet" href="styles.css">
    </head>

    Объяснение важных элементов:

    • <meta charset="UTF-8"> — указывает кодировку страницы, чаще всего это UTF-8, которая поддерживает множество языков.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0"> — задаёт масштаб страницы для правильного отображения на мобильных устройствах.
    • <link rel="stylesheet" href="styles.css"> — подключает внешний CSS-файл для стилизации страницы.
  3. Тег <body>

    4. Тег <body> содержит всё, что отображается на странице и с чем пользователь может взаимодействовать. Это "тело" документа, где располагаются тексты, изображения, ссылки, формы, видео и другие элементы.

    Пример содержимого <body>:

      
    <body>
      <h1>Привет, мир!</h1>
      <p>Это первый абзац текста.</p>
      <img src="example.jpg" alt="Описание изображения">
    </body>

    Основные элементы внутри <body>:

    • Заголовки (<h1> – <h6>) — они создают иерархию текста, где <h1> — самый важный заголовок.
    • Абзацы (<p>) — используются для оформления текста.
    • Изображения (<img>) — показывают графические файлы.
    • Ссылки (<a>) — позволяют переходить на другие страницы или сайты.
  4. Взаимодействие тегов

    5. Теги <html>, <head>, и <body> формируют основу любого HTML-документа:

    • <html> — охватывает весь документ.
    • <head> — содержит метаданные и не отображаемую информацию.
    • <body> — содержит видимый контент страницы.

    Пример полноценного HTML-документа:

      
    <!DOCTYPE html>
      <html lang="ru">
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Моя первая страница</title>
        <link rel="stylesheet" href="styles.css">
      </head>
      <body>
        <h1>Добро пожаловать на мою страницу!</h1>
        <p>Здесь вы узнаете много интересного.</p>
        <img src="image.jpg" alt="Пример изображения">
      </body>
    </html>
  5. Важные моменты:
    • DOCTYPE: всегда нужно начинать HTML-документ с <!DOCTYPE html>, чтобы указать браузеру, что это HTML5-документ.
    • Язык документа: задавай атрибут lang в теге <html>, чтобы улучшить доступность страницы и её правильное индексирование.
    • Мета-теги: они не влияют на внешний вид страницы, но играют важную роль в SEO и адаптивности.

    Теперь, понимая эту структуру, можно создавать страницы с правильной разметкой, заботясь как о видимом контенте, так и о "закулисной" информации, которая помогает браузерам и поисковикам лучше работать с твоим сайтом.
