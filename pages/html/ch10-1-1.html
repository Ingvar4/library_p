HTML

Структура HTML документа 
  
    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8">
        <link rel="shortcut icon" href="/images/image.jpg" type="image/x-icon">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta name="description" content="Описание страницы для поисковиков">
        <title>My First Webpage</title>
        <link rel="stylesheet" href="/styles/styles.css">
        <script src="/script/script.js" defer></script>
      </head>
      <body>
        <h1>Hello, World!</h1>
        <p>This is my first webpage.</p>
      </body>
    </html>
Теги HTML <br> - перенос на другую строку
<hr> - делает разделяющую полосу
<em> - курсив
<strong> - жирнит текст
<i> - Italic text
<mark> - Marked text
<small> - Smaller text
<del> - Deleted text
<ins> - Inserted text
<sub> - Subscript text
<sup> - Superscript text
<u> - подчёркивает текст
<blockquote> - цитирование
<abbr> - аббревиатура. Пример: Мы используем HTML для структурирования наших веб-документов.
<address> - элемент разметки контактных данных.
