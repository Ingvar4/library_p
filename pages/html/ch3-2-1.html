Тег <img>, атрибуты src, alt, width, height.

Тег <img> является основным HTML-элементом, который используется для вставки изображений на веб-страницы. Этот тег не имеет закрывающего тега, так как он является самозакрывающимся. Однако для работы с изображениями важно понимать, какие атрибуты связаны с <img>, а также как правильно их использовать для улучшения доступности, загрузки страниц и SEO.

1. Тег <img>

Тег <img> используется для отображения изображений на веб-странице. Он должен содержать как минимум один обязательный атрибут — src, который указывает путь к изображению.

Пример:

  
    <img src="images/photo.jpg" alt="Описание фотографии">

2. Атрибуты тега <img>

2.1 Атрибут src (source)

Атрибут src является обязательным и определяет путь к изображению, которое должно быть отображено на странице. Путь может быть:

  • Абсолютным — с указанием полного URL, ведущего к изображению.
    •   
    <img src="https://example.com/images/photo.jpg" alt="Описание фотографии">
  • Относительным — указывает на файл относительно текущего местоположения веб-страницы.
    •   
    <img src="images/photo.jpg" alt="Описание фотографии">

Если файл изображения не будет найден по указанному пути, вместо изображения на странице ничего не отобразится, или же пользователь увидит "сломанный" значок изображения.

2.2 Атрибут alt (alternative text)

Атрибут alt указывает альтернативный текст, который отображается, если изображение не загружается или недоступно. Этот атрибут также важен для доступности, так как используется экранными читалками для озвучивания текста вместо показа изображения.

  
    <img src="images/photo.jpg" alt="Мальчик играет с собакой">

Значимость атрибута alt:

  • SEO — поисковые системы учитывают текст в alt, что может положительно влиять на индексирование изображений.
  • Доступность — для людей с нарушениями зрения этот текст становится важным описанием изображения.
  • Загрузка изображений — в случае ошибки загрузки браузер покажет текст alt вместо изображения.

Рекомендуется делать alt коротким, но информативным и описывающим содержание изображения.

2.3 Атрибуты width и height

Атрибуты width и height определяют размеры изображения, задаваемые в пикселях. Эти атрибуты не являются обязательными, но их использование может ускорить загрузку страниц, так как браузер зарезервирует место для изображения до его полной загрузки.

  
    <img src="images/photo.jpg" alt="Описание фотографии" width="300" height="200">

Особенности работы с этими атрибутами:

  • Пропорции изображения — важно сохранять исходные пропорции изображения, чтобы оно не выглядело искажённым. Если указать только один атрибут, например, width, высота будет автоматически подстроена под пропорции изображения.
  • Адаптивность — в современных веб-сайтах для создания адаптивных дизайнов (когда сайт подстраивается под разные экраны) часто вместо фиксированных значений используют относительные размеры с помощью CSS. Например, width="100%", чтобы изображение занимало всю ширину контейнера.

2.4 Другие атрибуты

  • title — добавляет всплывающую подсказку, которая отображается при наведении курсора на изображение.
    •   
    <img src="images/photo.jpg" alt="Описание фотографии" title="Это подсказка">
  • loading="lazy" — откладывает загрузку изображения до момента, когда оно появится в зоне видимости пользователя. Это улучшает производительность страницы, особенно если на ней много изображений.
    •   
    <img src="images/photo.jpg" alt="Описание фотографии" loading="lazy">

3. Оптимизация изображений

Правильное использование тега <img> и его атрибутов помогает улучшить производительность сайта и восприятие его пользователями. Однако, помимо указанных атрибутов, есть ещё несколько факторов, которые стоит учитывать при работе с изображениями:

3.1 Размер файла изображения

Большие по размеру файлы могут замедлять загрузку страницы. Чтобы избежать этого:

  • Используй сжатие изображений (например, через инструменты вроде TinyPNG или ImageOptim).
  • Подбирай формат изображения в зависимости от его содержания. Например:
    • JPEG — лучше всего подходит для фотографий.
    • PNG — для изображений с прозрачностью или с высоким качеством.
    • WebP — современный формат сжатия, который уменьшает размер файла, сохраняя при этом качество.

3.2 Ретина-дисплеи и поддержка высоких разрешений

Для устройств с ретина-дисплеями (например, смартфонов или современных ноутбуков) требуется изображение с высоким разрешением. Это можно обеспечить, используя атрибут srcset:

  
    <img src="images/photo.jpg" srcset="images/photo-2x.jpg 2x" alt="Описание фотографии">

Здесь изображение photo-2x.jpg будет загружено на устройства с высоким разрешением экрана.

3.3 Адаптивные изображения

Для того чтобы адаптировать изображения под разные размеры экрана (например, мобильные устройства, планшеты, десктопы), используется атрибут srcset. Он позволяет загружать разные версии изображения в зависимости от размера экрана или плотности пикселей.

  
    <img src="images/photo-small.jpg" 
         srcset="images/photo-small.jpg 600w, images/photo-large.jpg 1200w" 
         sizes="(max-width: 600px) 100vw, 50vw" 
         alt="Описание фотографии">

Здесь браузер выберет изображение на основе ширины экрана устройства.

4. Атрибуты для повышения производительности и доступности

  • loading="lazy" — откладывает загрузку изображений, которые не находятся в поле зрения пользователя. Это улучшает скорость загрузки страницы.
    •   
    <img src="images/photo.jpg" alt="Описание фотографии" loading="lazy">
  • Доступность — всегда заполняй атрибут alt, чтобы пользователи, которые используют экранные читалки, могли понять контекст изображения. Также это помогает, если изображение не загрузится по какой-либо причине.

Заключение

Тег <img> является важным инструментом для отображения изображений на веб-странице. Правильное использование атрибутов src, alt, width, и height помогает сделать страницу более доступной, улучшить её производительность и пользовательский опыт. Не забывай об оптимизации изображений, чтобы страница загружалась быстрее, а контент был адаптивным для разных устройств.
