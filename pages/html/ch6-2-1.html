Теги <details>, <summary>, <dialog>, <progress>, <meter>.

Тег <details>

Тег <details> позволяет создавать скрываемый и раскрываемый блок информации, который можно развернуть/свернуть по клику. Часто используется для раскрывающихся описаний, инструкций и FAQ-разделов.

Основной синтаксис:

  
    <details>
      <summary>Подробнее о продукте</summary>
      <p>Это подробное описание продукта...</p>
    </details>

Результат:

Подробнее о продукте

Это подробное описание продукта...

Атрибуты:

  • open: По умолчанию раскрывает блок при загрузке страницы.

Как работает:

  1. <summary>: Содержит заголовок, на который пользователь кликает для раскрытия содержимого.
  2. Внутреннее содержимое: Всё, что находится внутри <details>, будет скрыто, пока блок не раскрыт.

Тег <summary>

Тег <summary> служит заголовком для <details>. Он показывает текст, на который кликает пользователь, чтобы развернуть скрытое содержимое.

Как использовать:

  1. Располагается внутри <details>.
  2. Один <summary> на <details>: Если несколько, будет использован только первый.

Пример:

  
    <details>
      <summary>Как начать?</summary>
      <p>Шаги для начала работы включают установку программного обеспечения...</p>
    </details>

Результат:

Как начать?

Шаги для начала работы включают установку программного обеспечения...

Тег <dialog>

Тег <dialog> — это элемент для создания диалоговых окон и модальных форм, встроенный в HTML5. Поддерживает элементы управления для интерактивного контента (форм, сообщений и уведомлений).

Основной синтаксис:

  
    <dialog id="myDialog">
      <p>Это сообщение в диалоговом окне.</p>
      <button onclick="document.getElementById('myDialog').close()">Закрыть</button>
    </dialog>

Результат:

Это сообщение в диалоговом окне.

Для того, чтобы отобразить данное диалоговое окно, был истользован атрибут open.

Атрибуты и методы:

  • open: Делает диалоговое окно видимым.
  • Метод show(): Показывает диалог, не делая его модальным.
  • Метод showModal(): Открывает модальное окно, блокирующее взаимодействие с другими элементами на странице.
  • Метод close(): Закрывает диалоговое окно.

Пример JavaScript:

  
    const dialog = document.getElementById('myDialog');
    dialog.showModal(); // Открывает окно как модальное
    dialog.close(); // Закрывает окно

Тег <progress>

Тег <progress> используется для отображения прогресса выполнения задачи, например, при загрузке или выполнении операции.

Основной синтаксис:

  
    <progress value="70" max="100">70%</progress>

Атрибуты:

  • value: Текущий прогресс (число).
  • max: Максимально возможное значение (по умолчанию — 1).

Использование:

  1. Динамическое обновление: Можно обновлять value в процессе выполнения задачи.
  2. Доступен стилизацией в CSS, хотя стилизация может быть ограниченной в некоторых браузерах.

Пример обновления с JavaScript:

  
    const progress = document.querySelector('progress');
    progress.value = 50; // Устанавливает значение на 50

Тег <meter>

Тег <meter> визуально отображает значение в определённом диапазоне, например, уровень заряда или оценку производительности.

Основной синтаксис:

  
    <meter value="0.7" min="0" max="1">70%</meter>

Атрибуты:

  • value: Текущее значение.
  • min: Минимально возможное значение.
  • max: Максимально возможное значение.
  • low, high, optimum: Позволяют настроить диапазоны, что поможет визуально выделить опасные и оптимальные уровни.

Пример:

  
    <meter value="50" min="0" max="100" low="20" high="80" optimum="60">50%</meter>

Эти теги позволяют легко создавать элементы интерфейса для отображения информации, уведомлений и прогресса.
