Базовые команды PostgreSQL (psql)

PostgreSQL предоставляет мощную утилиту командной строки под названием psql, которая позволяет управлять базами данных, выполнять SQL-запросы и многое другое. Ниже приведены основные команды для работы с PostgreSQL через psql.

1. Подключение к базе данных

Чтобы подключиться к серверу PostgreSQL и базе данных через psql, используйте следующую команду:

psql -U <имя_пользователя> -d <имя_базы_данных>

Например:

psql -U postgres -d my_database

Здесь -U указывает имя пользователя, а -d — имя базы данных. Если база данных запущена на удалённом сервере, добавьте флаг -h для указания хоста:

psql -U postgres -d my_database -h 192.168.1.10

2. Выход из psql

Чтобы выйти из консоли psql, просто выполните команду:

\q

3. Список баз данных

Чтобы просмотреть список всех баз данных на сервере PostgreSQL, выполните команду:

\l

Вы увидите таблицу с именами баз данных, пользователями и кодировками.

4. Подключение к другой базе данных

Внутри консоли psql можно переключиться на другую базу данных, используя команду:

\c <имя_базы_данных>

Например:

\c another_database

5. Список таблиц в базе данных

Чтобы просмотреть все таблицы в текущей базе данных, выполните команду:

\dt

Эта команда отобразит список всех таблиц, их владельцев и типов.

6. Просмотр структуры таблицы

Чтобы посмотреть структуру (схему) конкретной таблицы, используйте команду:

\d <имя_таблицы>

Например:

\d users

Эта команда отобразит информацию о столбцах таблицы, их типах данных и ограничениях.

7. Выполнение SQL-запросов

Через psql можно выполнять любые SQL-запросы. Например, чтобы выбрать все записи из таблицы users, выполните запрос:

SELECT * FROM users;

Также можно выполнять другие команды SQL, такие как вставка, обновление и удаление данных.

8. Создание новой базы данных

Чтобы создать новую базу данных через консоль psql, выполните команду:

CREATE DATABASE <имя_базы_данных>;

Например:

CREATE DATABASE new_database;

9. Удаление базы данных

Чтобы удалить базу данных, используйте команду:

DROP DATABASE <имя_базы_данных>;

Например:

DROP DATABASE old_database;

10. Экспорт данных в файл

Чтобы экспортировать данные из базы данных в файл, используйте команду pg_dump:

pg_dump -U <имя_пользователя> <имя_базы_данных> > <файл>.sql

Например:

pg_dump -U postgres my_database > backup.sql

11. Импорт данных из файла

Чтобы импортировать данные из файла в базу данных, используйте команду:

psql -U <имя_пользователя> -d <имя_базы_данных> -f <файл>.sql

Например:

psql -U postgres -d my_database -f backup.sql

Заключение

Утилита psql предоставляет мощные возможности для управления базами данных PostgreSQL. С её помощью можно выполнять SQL-запросы, управлять базами данных, просматривать структуры таблиц и экспортировать данные. Освоение базовых команд psql поможет вам уверенно работать с PostgreSQL на уровне терминала.
