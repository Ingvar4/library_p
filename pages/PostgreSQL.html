План обучения PostgreSQL

Этап 1: Введение в базы данных и PostgreSQL

  1. Что такое базы данных?
    • Понятие реляционных баз данных.
    • Основные компоненты баз данных (таблицы, строки, столбцы).
    • Введение в SQL (язык структурированных запросов).
  2. Установка PostgreSQL
    • Установка PostgreSQL на Windows, macOS или Linux.
    • Установка графического интерфейса pgAdmin (для удобной работы с базой).
    • Настройка окружения.
  3. Основные команды PostgreSQL
    • Подключение к базе данных через терминал или pgAdmin.
    • Базовые команды PostgreSQL (psql, подключение к базам).

Этап 2: Основы SQL

  1. Создание баз данных и таблиц
    • Команда CREATE DATABASE.
    • Команда CREATE TABLE.
    • Типы данных в PostgreSQL (INTEGER, VARCHAR, DATE, BOOLEAN и т.д.).
  2. Операции CRUD
    • Create: Вставка данных с помощью командыINSERT.
    • Read: Чтение данных с помощью команды SELECT (простые запросы).
    • Update: Обновление данных с помощью команды UPDATE.
    • Delete: Удаление данных с помощью команды DELETE.
  3. Операции с фильтрацией и сортировкой
    • Использование условий с WHERE.
    • Фильтрация с LIKE, IN, BETWEEN.
    • Сортировка данных с помощью ORDER BY.
    • Лимитирование результатов с помощью LIMIT.

Этап 3: Углубление в запросы

  1. Агрегация данных
    • Функции агрегации (COUNT, SUM, AVG, MAX, MIN).
    • Группировка данных с помощью GROUP BY.
    • Фильтрация агрегированных данных с HAVING.
  2. Объединение данных
    • Использование различных видов JOIN (INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN).
    • Работа с несколькими таблицами.
  3. Подзапросы
    • Вложенные запросы (подзапросы).
    • Использование ключевых слов EXISTS, ANY, ALL.

Этап 4: Работа с индексами и оптимизация

  1. Индексы
    • Зачем нужны индексы и как они работают.
    • Создание индексов с помощью команды CREATE INDEX.
    • Разновидности индексов (B-tree, Hash).
  2. Оптимизация запросов
    • Введение в EXPLAIN и анализ планов выполнения запросов.
    • Как улучшить производительность запросов.
    • Избежание избыточных запросов.

Этап 5: Работа с транзакциями

  1. Транзакции
    • Основные команды транзакций: BEGIN, COMMIT, ROLLBACK.
    • Понятие ACID (атомарность, согласованность, изоляция, долговечность).
    • Уровни изоляции транзакций.
  2. Ограничения и связи
    • Первичные ключи (PRIMARY KEY).
    • Внешние ключи (FOREIGN KEY).
    • Ограничения уникальности (UNIQUE).
    • Ограничения проверки (CHECK).

Этап 6: Администрирование PostgreSQL

  1. Пользователи и права доступа
    • Создание пользователей и ролей.
    • Назначение прав доступа (GRANT, REVOKE).
    • Управление правами на уровне таблиц и баз данных.
  2. Резервное копирование и восстановление
    • Использование pg_dump для резервного копирования.
    • Восстановление данных с помощью pg_restore.
  3. Мониторинг производительности
    • Основные инструменты для мониторинга PostgreSQL.
    • Понятие блокировок и deadlock'ов.

Этап 7: Дополнительные темы

  1. Расширенные возможности PostgreSQL
    • Работа с JSON-данными.
    • Хранение и манипуляция массивами.
    • Встроенные функции и создание пользовательских функций.
  2. Миграция данных
    • Импорт и экспорт данных (CSV, JSON).
    • Использование COPY для массовой вставки данных.

Этап 8: Практика и проекты

  1. Создание реального проекта
    • Проект: создание базы данных для простого веб-приложения.
    • Проект: анализ данных с помощью агрегационных запросов и JOIN'ов.
  2. Работа с ORM (Object-Relational Mapping)
    • Использование PostgreSQL с популярными фреймворками (например, Django, Sequelize).
