Базовые операции с числами и строками в JavaScript

Работа с числами и строками — это одна из ключевых тем в программировании на JavaScript. Понимание того, как эффективно использовать эти типы данных, является основой для написания любого кода. Рассмотрим базовые операции с числами и строками в JavaScript.

1. Работа с числами

В JavaScript числа представлены одним типом данных number, который включает как целые числа, так и числа с плавающей запятой. Есть также специальное значение для бесконечности (Infinity) и значение, обозначающее не число (NaN).

Основные операции с числами:

  • Сложение: +
  • Вычитание: -
  • Умножение: *
  • Деление: /
  • Возведение в степень: **
  • Остаток от деления: %
  
    let a = 10;
    let b = 3;

    console.log(a + b); // 13
    console.log(a - b); // 7
    console.log(a * b); // 30
    console.log(a / b); // 3.3333333333333335
    console.log(a ** b); // 1000
    console.log(a % b); // 1

Инкремент и декремент:

  • ++ увеличивает значение переменной на 1.
  • -- уменьшает значение переменной на 1.
  
    let count = 5;
    count++;
    console.log(count); // 6

    count--;
    console.log(count); // 5

Работа с бесконечностью и NaN:

  • Infinity возникает при делении на ноль или при выполнении других операций, которые выходят за пределы допустимого диапазона чисел.
  • NaN (Not-a-Number) — это результат некорректной математической операции, например, деления строки на число.
  
    console.log(1 / 0); // Infinity
    console.log("hello" / 2); // NaN

2. Работа со строками

Строки в JavaScript — это последовательности символов, заключенные в одинарные ('...'), двойные ("..."), или обратные кавычки (`...`). Строки неизменяемы, то есть после создания строки вы не можете изменить её содержимое, но можете создать новую строку на основе старой.

Основные операции со строками:

  • Конкатенация: Операция объединения строк с использованием оператора +.
  
    let greeting = "Hello, ";
    let name = "Alice";
    let message = greeting + name;
    console.log(message); // "Hello, Alice"

Шаблонные строки:

С использованием обратных кавычек и вставкой значений через ${...}.

  
    let age = 30;
    let info = `Alice is ${age} years old.`;
    console.log(info); // "Alice is 30 years old."

Длина строки:

Получить длину строки можно с помощью свойства length.

  
    let text = "Hello, world!";
    console.log(text.length); // 13

Доступ к символам строки:

Вы можете получить доступ к отдельным символам строки, используя индекс.

  
    let text = "JavaScript";
    console.log(text[0]); // "J"
    console.log(text[text.length - 1]); // "t"

Методы для работы со строками:

  • toUpperCase() и toLowerCase() для изменения регистра.
  • indexOf() для поиска подстроки.
  • slice() для извлечения части строки.
  • split() для разбиения строки на массив по заданному разделителю.
  
    let text = "Hello, World!";
    console.log(text.toUpperCase()); // "HELLO, WORLD!"
    console.log(text.toLowerCase()); // "hello, world!"

    console.log(text.indexOf("World")); // 7

    console.log(text.slice(0, 5)); // "Hello"
    console.log(text.split(", ")); // ["Hello", "World!"]

Заключение

Работа с числами и строками — это фундаментальная часть программирования на JavaScript. Знание этих основ поможет вам писать более эффективный код и лучше понимать, как обрабатываются данные в вашем приложении.
