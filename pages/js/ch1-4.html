Введение в синтаксис JavaScript (переменные, типы данных)

1. Переменные

Переменные в JavaScript — это контейнеры для хранения данных. В JavaScript существует три способа объявления переменных:

  • var: Исторический способ объявления переменных. Переменные, объявленные с помощью var, имеют функциональную область видимости (если они объявлены внутри функции) или глобальную область видимости (если объявлены вне функции). Переменные, объявленные с var, могут быть переобъявлены и перезаписаны.
  • let: Современный способ объявления переменных. let имеет блочную область видимости, то есть переменная видима только в пределах блока кода, в котором она объявлена. Переменные, объявленные с помощью let, могут быть перезаписаны, но не могут быть переобъявлены в той же области видимости.
  • const: Используется для объявления констант, значения которых не могут быть изменены после присвоения. const также имеет блочную область видимости. Переменные, объявленные с const, должны быть инициализированы в момент их объявления.

Примеры:

  
    var name = "John"; // Объявление переменной с использованием var
    let age = 25; // Объявление переменной с использованием let
    const country = "USA"; // Объявление переменной с использованием const

2. Типы данных

В JavaScript существует несколько основных типов данных, которые можно разделить на примитивные и ссылочные.

Примитивные типы данных:

  • String: Представляет текстовые данные. Строки заключаются в одинарные или двойные кавычки.
    •   
    let greeting = "Hello, world!";
  • Number: Представляет как целые числа, так и числа с плавающей запятой.
    •   
    let age = 30;code 
    let pi = 3.14;
  • Boolean: Представляет логические значения true (истина) или false (ложь)
    •   
    let isStudent = true;
  • Null: Специальное значение, которое означает отсутствие какого-либо значения.
    •   
    let emptyValue = null;
  • Undefined: Переменная, которая была объявлена, но ей не было присвоено значение.
    •   
    let undefinedValue;
  • Symbol: Уникальное и неизменное значение, которое может быть использовано как идентификатор для свойств объектов.
    •   
    let symbol = Symbol('description');

Ссылочные типы данных:

  • Object: Комплексный тип данных, который используется для хранения коллекций данных или более сложных сущностей.
    •   
    let person = {
      name: "John",
      age: 30,
      isStudent: false
    };
  • Array: Тип данных, который представляет собой упорядоченный список значений.
    •   
    let numbers = [1, 2, 3, 4, 5];
  • Function: Объект, представляющий функцию, которая может быть вызвана.
    •   
    function greet() {
      console.log("Hello!");
    }

3. Динамическая типизация

JavaScript — это язык с динамической типизацией, что означает, что тип переменной определяется автоматически в момент выполнения программы, а не в момент объявления переменной. Это также означает, что переменная может менять свой тип в процессе выполнения программы.

  
    let variable = 42;    // variable имеет тип Number
    variable = "Hello";   // Теперь variable имеет тип String

4. Операции с переменными и типами данных

  • Конкатенация строк: Складывание строк с помощью оператора "+".
    •   
    let firstName = "John";
    let lastName = "Doe";
    let fullName = firstName + " " + lastName; // "John Doe"
  • Арифметические операции: Стандартные математические операции, такие как сложение, вычитание, умножение и деление.
    •   
    let sum = 10 + 5; // 15
    let product = 4 * 3; // 12
  • Логические операции: Операции, которые возвращают логическое значение "true" или "false".
    •   
    let isAdult = age > 18; // true, если age больше 18

Понимание переменных и типов данных является основой для дальнейшего изучения JavaScript. Эти концепции используются повсеместно и позволяют более эффективно писать код, решать задачи и разрабатывать интерактивные веб-приложения.
