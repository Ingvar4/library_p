Изменение стилей и содержимого элементов DOM

Возможность динамически изменять стили и содержимое элементов DOM с помощью JavaScript — одна из ключевых возможностей для создания интерактивных веб-страниц. Давайте подробнее рассмотрим, как это можно делать.

  1. Изменение текстового содержимого

    2. Существует несколько способов изменить текст внутри HTML-элемента:

    1.1 textContent

    Свойство textContent позволяет получить или изменить текстовое содержимое элемента, включая текст всех его потомков.

    Пример:

      
    let header = document.getElementById("main-header");
    header.textContent = "Новый заголовок";

    Этот код изменяет текст элемента с id "main-header" на "Новый заголовок". Важно отметить, что textContent заменяет весь текст внутри элемента, включая пробелы и текст внутри дочерних элементов.

    1.2 innerHTML

    Свойство innerHTML позволяет получить или изменить HTML-содержимое элемента, включая вложенные элементы и текст.

    Пример:

      
    let container = document.querySelector(".container");
    container.innerHTML = "<h2>Заголовок уровня 2</h2><p>Новый текст параграфа</p>";

    Этот код заменяет всё содержимое элемента с классом container на новый заголовок и параграф.

    Отличие от textContent: innerHTML позволяет вставлять HTML-разметку, тогда как textContent работает только с текстом.

    1.3 innerText

    Свойство innerText похоже на textContent, но оно учитывает стили и отображает текст так, как он виден пользователю, с учетом CSS. Это значит, что innerText игнорирует текст, скрытый с помощью CSS, тогда как textContent возвращает весь текст, включая скрытый.

    Пример:

      
    let paragraph = document.querySelector(".paragraph");
    paragraph.innerText = "Изменённый текст параграфа";

    Этот код изменяет видимый текст параграфа на "Изменённый текст параграфа".

  2. Изменение стилей элементов

    3. Для изменения стилей элементов DOM используется свойство style, которое позволяет изменять любые CSS-свойства, примененные к элементу.

    2.1 Изменение отдельных стилей

    Каждое CSS-свойство можно установить как отдельное свойство объекта style. Для этого используется camelCase-нотация, где каждое слово в имени CSS-свойства, кроме первого, пишется с заглавной буквы.

    Пример:

      
    let header = document.getElementById("main-header");
    header.style.color = "blue";
    header.style.fontSize = "24px";
    header.style.backgroundColor = "lightgray";

    Этот код изменяет цвет текста заголовка на синий, увеличивает размер шрифта до 24 пикселей и устанавливает светло-серый цвет фона.

    2.2 Изменение нескольких стилей одновременно

    Для применения нескольких стилей одновременно можно использовать строку CSS-кода.

    Пример:

      
    header.style.cssText = "color: blue; font-size: 24px; background-color: lightgray;";

    Этот код делает то же самое, что и предыдущий, но в одной строке.

    2.3 Управление классами через classList

    Вместо того чтобы изменять отдельные стили, часто удобнее управлять классами элементов. Для этого используется свойство classList, которое предоставляет методы add, remove, toggle, contains.

    Пример:

      
    let button = document.querySelector("button");
    button.classList.add("active");     // Добавляет класс "active"
    button.classList.remove("inactive"); // Удаляет класс "inactive"
    button.classList.toggle("highlight"); // Переключает класс "highlight"

    Этот код управляет классами кнопки, добавляя, удаляя и переключая их.

    2.4 Изменение стилей через CSS-переменные

    Если на странице используются CSS-переменные (также называемые пользовательскими свойствами), их можно изменять через JavaScript, что позволяет гибко управлять стилями на уровне всего документа.

    Пример:

      
    document.documentElement.style.setProperty('--main-bg-color', 'coral');

    Этот код изменяет значение CSS-переменной --main-bg-color, что автоматически обновит все стили на странице, зависящие от этой переменной.

  3. Изменение атрибутов элементов

    4. Иногда нужно изменить атрибуты HTML-элементов, такие как src, href, alt, и другие. Это делается с помощью методов setAttribute, getAttribute, removeAttribute.

    Пример:

      
    let image = document.querySelector("img");
    image.setAttribute("src", "new-image.jpg"); // Меняет путь к изображению
    image.setAttribute("alt", "Новое изображение"); // Меняет текстовое описание

    Этот код изменяет путь к изображению и текстовое описание для элемента img.

Заключение

Изменение стилей и содержимого элементов DOM — это мощный инструмент для создания динамических и интерактивных веб-страниц. Владение этими методами позволяет разработчикам гибко управлять внешним видом и содержанием страницы в зависимости от действий пользователя или других условий, что делает веб-приложения более отзывчивыми и удобными в использовании.
