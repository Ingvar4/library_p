Тернарный оператор в JavaScript

Тернарный оператор (условный оператор) в JavaScript — это короткий способ записи условного выражения, который заменяет собой простую конструкцию if...else. Он используется для выполнения одного из двух выражений в зависимости от условия.

  1. Синтаксис тернарного оператора

    2. Тернарный оператор имеет следующий синтаксис:

      
    условие ? выражение1 : выражение2;
    • условие — выражение, которое возвращает либо true, либо false.
    • выражение1 — выполняется, если условие истинно (true).
    • выражение2 — выполняется, если условие ложно (false).
  2. Простой пример использования

    3. Допустим, нужно присвоить переменной status значение "Совершеннолетний", если возраст (age) больше или равен 18, и "Несовершеннолетний" в противном случае.

    С использованием тернарного оператора это можно сделать следующим образом:

      
    let age = 20;
    let status = age >= 18 ? "Совершеннолетний" : "Несовершеннолетний";

    console.log(status); // Вывод: "Совершеннолетний"

    Здесь, если age больше или равно 18, переменной status присваивается значение "Совершеннолетний". Если age меньше 18, переменной присваивается "Несовершеннолетний".

  3. Пример с использованием функции

    4. Тернарный оператор можно использовать для возврата значений из функции:

      
    function checkNumber(number) {
      return number > 0 ? "Положительное" : "Отрицательное или Ноль";
    }
  
    let result = checkNumber(5);
    console.log(result); // Вывод: "Положительное"

    Здесь функция checkNumber возвращает строку "Положительное", если число больше нуля, и "Отрицательное или Ноль" в противном случае.

  4. Вложенные тернарные операторы

    5. Вложенные тернарные операторы позволяют проверять несколько условий, хотя использование их таким образом может сделать код менее читаемым.

    Пример:

      
    let score = 85;
    let grade = score >= 90 ? "Отлично" : score >= 75 ? "Хорошо" : "Попробуйте еще раз";

    console.log(grade); // Вывод: "Хорошо"

    Здесь, если score больше или равен 90, переменной grade присваивается "Отлично". Если score меньше 90, но больше или равен 75, присваивается "Хорошо". В противном случае присваивается "Попробуйте еще раз".

  5. Когда использовать тернарный оператор?

    6. Тернарный оператор удобен для коротких условий, когда нужно быстро присвоить значение или вернуть результат. Однако для более сложных логических условий или когда требуется выполнить несколько операций, лучше использовать стандартные конструкции if...else, чтобы сохранить читабельность кода.

Заключение

Тернарный оператор — это мощный и удобный инструмент для упрощения условий в JavaScript. Он позволяет сократить код, делая его более лаконичным и иногда более удобным для чтения. Тем не менее, важно помнить о читабельности, особенно при использовании вложенных тернарных операторов.
