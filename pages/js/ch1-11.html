Практика: Написание простых функций и выполнение арифметических операций в JavaScript

  1. Что такое простая функция?

    2. Простая функция в JavaScript — это функция, которая выполняет одну конкретную задачу. Она может принимать один или несколько параметров, выполнять какие-то операции и, при необходимости, возвращать результат. Простые функции — это основа программирования, так как они позволяют разделить сложные задачи на более мелкие и управляемые части.

  2. Примеры простых функций с арифметическими операциями

    3. Давайте рассмотрим несколько примеров простых функций, которые выполняют базовые арифметические операции.

    Пример 1: Функция сложения
    Эта функция принимает два числа и возвращает их сумму.

      
    function add(a, b) {
      return a + b;
    }
  
    let result = add(3, 5); // result = 8
    console.log(result);

    Пример 2: Функция вычитания
    Эта функция принимает два числа и возвращает разницу между ними.

      
    function subtract(a, b) {
      return a - b;
    }
  
    let result = subtract(10, 4); // result = 6
    console.log(result);

    Пример 3: Функция умножения
    Функция, которая принимает два числа и возвращает их произведение.

      
    function multiply(a, b) {
      return a * b;
    }
  
    let result = multiply(6, 7); // result = 42
    console.log(result);

    Пример 4: Функция деления
    Эта функция принимает два числа и возвращает результат их деления.

      
    function divide(a, b) {
      if (b !== 0) {
          return a / b;
      } else {
          return "Ошибка: деление на ноль";
      }
    }
  
    let result = divide(15, 3); // result = 5
    console.log(result);
  
    let error = divide(10, 0); // error = "Ошибка: деление на ноль"
    console.log(error);
  3. Более сложные примеры: комбинация арифметических операций

    4. Теперь, когда мы знаем, как писать простые функции для выполнения отдельных операций, можно создавать более сложные функции, которые будут сочетать несколько арифметических операций.

    Пример 5: Функция расчета среднего значения
    Эта функция принимает три числа и возвращает их среднее арифметическое.

      
    function calculateAverage(a, b, c) {
      let sum = add(a, b) + c; // Используем ранее созданную функцию сложения
      return sum / 3;
    }
  
    let average = calculateAverage(4, 8, 12); // average = 8
    console.log(average);

    Пример 6: Функция расчета процента от числа

    Функция принимает два числа: общее значение и процент, который нужно вычислить, и возвращает результат.

      
    function calculatePercentage(total, percentage) {
      return (total * percentage) / 100;
    }
  
    let percentageValue = calculatePercentage(200, 15); // percentageValue = 30
    console.log(percentageValue);
  4. Практика написания и использования функций

    5. Чтобы уверенно работать с функциями, полезно регулярно практиковаться, решая небольшие задачи, которые включают арифметические операции. Например:

    • Напишите функцию, которая принимает два числа и возвращает их остаток от деления.
    • Создайте функцию, которая рассчитывает периметр прямоугольника, зная его длину и ширину.
    • Напишите функцию, которая принимает три числа и возвращает максимальное из них.

    Пример 7: Функция для вычисления остатка от деления

      
    function modulus(a, b) {
      return a % b;
    }
  
    let remainder = modulus(17, 5); // remainder = 2
    console.log(remainder);

    Пример 8: Функция расчета периметра прямоугольника

      
    function calculatePerimeter(length, width) {
      return 2 * (length + width);
    }
  
    let perimeter = calculatePerimeter(5, 10); // perimeter = 30
    console.log(perimeter);

    Пример 9: Функция для нахождения максимального числа

      
    function findMax(a, b, c) {
      return Math.max(a, b, c);
    }
  
    let maxNumber = findMax(12, 7, 9); // maxNumber = 12
    console.log(maxNumber);

Заключение

Написание простых функций с арифметическими операциями — это основа для понимания более сложных концепций программирования. Практика таких задач поможет вам лучше понять, как работают функции, и как они могут быть использованы для решения реальных проблем. Важно начать с простых примеров и постепенно переходить к более сложным задачам, комбинируя несколько функций и операций в одном решении.
