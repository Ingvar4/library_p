Массивы в js

Структура и создание

Массив — это список, где хранятся другие значения. Для создания массива используются квадратные скобки [];
Чтобы создать массив со значениями, нужно перечислить эти значения внутри квадратных скобок, разделяя их запятыми. Отдельные значения, хранящиеся в массиве, называют элементами. В данном примере все элементы будут строковыми, поэтому записываются в кавычках.

var elementsBox = ["firstElement", "secondElement", "thirdElement", "fourthElement"];

Чтобы получить доступ к элементам массива, используют квадратные скобки с индексом нужного элемента, как в примере:

elementsBox [0];
'firstElement'
elementsBox [1];
'secondElement'

Индекс — это номер элемента, в котором хранится значение. Аналогично символам в строке, первому элементу массива соответствует индекс 0, второму — 1, третьему — 2 и т. д.

Создание и изменение элементов

Чтобы заменить содержимое 1го элемента:
elementsBox [0] = "mainElement";
 получится:
elementsBox;
['mainElement', 'secondElement', 'thirdElement', 'fourthElement']

С помощью индексов также можно добавлять в массив элементы. Например, вот как создать массив elementsBox, задавая каждый элемент через квадратные скобки:
var elementsBox = [];
elementsBox [0] = "firstElement";
elementsBox [1] = "secondElement";
elementsBox [2] = "thirdElement";
elementsBox [3] = "fourthElement";
получаем:
elementsBox;
['firstElement', 'secondElement', 'thirdElement', 'fourthElement']

В массив можно так же добавить элемент с любым индексом.
elementsBox [20] = "twentiethElement"
получаем:
elementsBox;
['firstElement', 'secondElement', 'thirdElement', 'fourthElement', empty × 16, 'twentiethElement']
Элементы между индексами 3 и 20 получат значение undefined. При печати массива Chrome сообщает количество этих undefined-элементов, а не выводит каждый из них по отдельности.

Разные типы данных в одном массиве

Не обязательно, чтобы все элементы массива были одного типа. Например, вот массив, в котором хранится число (3), строка ("динозавры"), массив (["трицератопс", "стегозавр", 3627.5]) и еще одно число (10):

var dinosaursAndNumbers = [3, "динозавры", ["трицератопс", "стегозавр", 3627.5], 10];

Чтобы обратиться к элементам массива, вложенного в другой массив, нужно использовать вторую пару квадратных скобок. Например, если команда dinosaursAndNumbers[2]; вернет весь вложенный массив, то dinosaursAndNumbers[2][0]; — лишь первый элемент этого вложенного массива ("трицератопс").

dinosaursAndNumbers[2];
["трицератопс", "стегозавр", 3627.5]
dinosaursAndNumbers[2][0];
"трицератопс"

Работа с массивами. Свойства и методы.

Длинна массива

Cвойство .length хранит количество элементов в массиве. Чтобы узнать длину массива, просто добавьте .length после его имени.

Создаём массив:

var elementsBox = ["firstElement", "secondElement", "thirdElement", "fourthElement"];

Чтобы узнать длину этого массива, добавим .length к elementsBox:

elementsBox.length;
4

JavaScript сообщает, что в массиве 4 элемента, и мы знаем, что их индексы — 0, 1, 2 и 3. Отсюда следует полезное наблюдение: последний индекс массива всегда на единицу меньше длины этого массива. Это значит, что есть простой способ получить последний элемент массива, какой бы ни была его длина:

elementsBox [elementsBox.length - 1];
'fourthElement'

Мы попросили JavaScript вернуть элемент из нашего массива, но вместо числового индекса ввели в квадратных скобках выражение: длина массива минус 1. JavaScript нашел свойство elementsBox.length со значением 4, вычел 1, получив 3, и наконец вернул элемент с индексом 3 — это и есть последний элемент, "fourthElement".

Добавление элементов в массив

Чтобы добавить элемент к концу массива, можно воспользоваться методом .push ("");

var elementsBox = ["firstElement", "secondElement", "thirdElement", "fourthElement"];
elementsBox;
['firstElement', 'secondElement', 'thirdElement', 'fourthElement']
elementsBox.push ("fifthElement");
elementsBox;
['firstElement', 'secondElement', 'thirdElement', 'fourthElement', 'fifthElement']
  1. Создали массив
  2. Вызвали его содержимое
  3. Непосредственно содержимое массива
  4. Добавили 5й элемент в массив
  5. Вызвали содержимое массива
  6. Непосредственно содержимое массива

Чтобы добавить элемент в начало массива, можно воспользоваться методом .unshift("");

elementsBox.unshift ("zeroElement");
elementsBox;
['zeroElement', 'firstElement', 'secondElement', 'thirdElement', 'fourthElement', 'fifthElement']

Удаление элементов массива

Убрать из массива последний элемент можно, добавив к его имени .pop(); Метод pop делает сразу два дела: удаляет последний элемент из массива и возвращает этот элемент в виде значения. 

elementsBox;
['zeroElement', 'firstElement', 'secondElement', 'thirdElement', 'fourthElement', 'fifthElement']
var deletedElements = elementsBox.pop ();
deletedElements;
'fifthElement'
elementsBox;
['zeroElement', 'firstElement', 'secondElement', 'thirdElement', 'fourthElement']
elementsBox.pop ();
'fourthElement'
elementsBox;
['zeroElement', 'firstElement', 'secondElement', 'thirdElement']
elementsBox.unshift (deletedElements);
5
elementsBox;
['fifthElement', 'zeroElement', 'firstElement', 'secondElement', 'thirdElement']
  1. Вызвали содержимое массива
  2. Его cодержимое
  3. Создали переменную deletedElements и сохранили в неё удалённый из elementsBox последний элемент
  4. Вызвали содержимое переменной
  5. Её содержимое
  6. Вызвали содержимое массива
  7. Его содержимое
  8. Удалили последний элемент из массива
  9. Возврат значения удалённого элемента
  10. Вызвали содержимое массива
  11. Его содержимое
  12. Добавили в начало массива элемент из переменной deletedElements
  13. Текущее количество элементов в массиве
  14. Вызвали содержимое массива
  15. Его содержимое

Убрать из массива первый элемент можно, добавив к его имени .shift();

elementsBox;
['fifthElement', 'zeroElement', 'firstElement', 'secondElement', 'thirdElement']
elementsBox.shift ();
'fifthElement'
elementsBox;
['zeroElement', 'firstElement', 'secondElement', 'thirdElement']

Объединение массивов

Чтобы «склеить» два массива, создав таким образом новый массив, используйте команду firstArray.concat(otherArray);
Метод .concat создает массив, в котором элементы из firstArray будут расположены перед элементами из otherArray.

var elementsBox = ["firstElement", "secondElement", "thirdElement", "fourthElement"];
var otherElBox = ["fifthElement", "sixthElement"];
var allElements = elementsBox.concat(otherElBox);
allElements;
['firstElement', 'secondElement', 'thirdElement', 'fourthElement', 'fifthElement', 'sixthElement']
elementsBox
['firstElement', 'secondElement', 'thirdElement', 'fourthElement']
otherElBox
['fifthElement', 'sixthElement']

Создали два массива, склеили, проверили. Можно объединять несколько массивов через запятую в скобках.
firstArray.concat(otherArray, otherArray);

Поиск индекса элемента в массиве

Чтобы выяснить, какой у определенного элемента индекс в массиве, используйте .indexOf("элемент");

var colors = ["красный", "зеленый", "синий"];
colors.indexOf("синий");
2
colors.indexOf("зеленый");
1

Если элемент встречается в массиве больше чем один раз, indexOf вернет индекс того элемента, который находится ближе к началу массива

var insects = ["Пчела", "Муравей", "Пчела", "Пчела", "Муравей"];
insects.indexOf("Пчела");
0

Превращаем массив в строку

Воспользовавшись методом .join(); можно соединить все элементы массива в одну большую строку.

var elementsBox = ["firstElement", "secondElement", "thirdElement", "fourthElement"];
elementsBox.join();
'firstElement,secondElement,thirdElement,fourthElement'

Можно воскользоваться разделителем, к примеру .join(" - "); .join("*"); .join(" и ");

elementsBox.join(" - ");
'firstElement - secondElement - thirdElement - fourthElement'
elementsBox.join(" * ");
'firstElement * secondElement * thirdElement * fourthElement'
elementsBox.join(" и ");
'firstElement и secondElement и thirdElement и fourthElement'

Так же можно использовать .join(" "); с пробелом внутри кавычек чтобы непосредственно создать разделитель:

elementsBox.join(" ");
'firstElement secondElement thirdElement fourthElement'

Использование Math.random()

Math.random() генерирует случайные числа:

Math.random()
0.4290244147728972
Math.random()
0.5484079001690576
Math.random()
0.9260613977145511

Важно помнить, что Math.random() всегда возвращает число меньше 1, то есть никогда не возвращает собственно 1. Если вам нужно число побольше, просто умножьте полученное из метода Math.random() значение на подходящий коэффициент. Например, если нужно случайное число от 0 до 10, умножьте Math.random() на 10:

Math.random() * 10;
8.053146225265666
Math.random() * 10;
7.8434561749153335
Math.random() * 10;
8.348755628569677

Округление с помощью Math.floor()

И все же эти случайные значения нельзя использовать как индексы в массиве, поскольку индексы должны быть целыми числами, а не десятичными дробями. Чтобы исправить этот недостаток, нужен метод Math.floor(); округляющий число до ближайшего снизу целого значения (по сути, он просто отбрасывает все знаки после запятой).

Math.floor(3.7463463);
3
Math.floor(9.9999);
9
Math.floor(0.793423451963426);
0
