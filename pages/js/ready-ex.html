Кнопка "Subscribe"

При нажатии запускается функция, которая меняет надпись на "Subscribed" и обратно.

  
    <button onclick="subscribe();" class="js-subscribe-button">Subscribe</button>
  

  
    function subscribe () {
      const buttonElement = document.querySelector('.js-subscribe-button')
      if (buttonElement.innerHTML === 'Subscribe') {
        buttonElement.innerHTML = 'Subscribed';
      } else {
        buttonElement.innerHTML = 'Subscribe';
      }
    }
  1. Кнопке задаём свойсво onclick и класс "js-subscribe-button"
  2. Создаём функцию subscribe, а внутри неё переменную buttonElement, которая через метод document.querySelector() возвращает первый элемент, который соответствует указанному селектору или группе селекторов.
  3. Далее создаём условие с помощью if...else:
    • if: если (проверяем с помощью считывания методом .innerHTML истинно ли условие, что содержимое элемента переменной buttonElement равно 'Subscribe'?) Если да, то с помощью метода .innerHTML меняем содержимое на "Subscribed"
    • else: в противном случае меняем содержимое на 'Subscribe' (если "if" не true)
