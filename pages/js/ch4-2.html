Поиск и манипуляция элементами DOM (getElementById, querySelector)

Взаимодействие с элементами DOM — одна из ключевых задач при разработке веб-приложений. Методы getElementById, querySelector и другие позволяют находить элементы на странице и изменять их содержимое, стиль и атрибуты. Рассмотрим эти методы подробнее.

  1. Поиск элементов DOM

    2. Поиск элементов DOM осуществляется с помощью методов, которые позволяют находить элементы по различным критериям, таким как id, класс, тег и CSS-селектор.

    1.1 getElementById

    Метод getElementById — это самый простой и быстрый способ найти элемент на странице по его уникальному идентификатору (id). Этот метод возвращает один элемент, так как id должен быть уникальным на странице.

    Пример:

      
    ⁄⁄HTML
    <div id="main-header">Заголовок</div>
  

      
    // JavaScript
    let header = document.getElementById("main-header");
    console.log(header.textContent); // "Заголовок"

    В этом примере метод getElementById находит элемент с id "main-header" и возвращает его.

    1.2 querySelector

    Метод querySelector — это более универсальный способ поиска элементов. Он принимает в качестве аргумента любой CSS-селектор и возвращает первый элемент, который соответствует этому селектору. Если ни один элемент не соответствует селектору, возвращается null.

    Пример:

      
    ⁄⁄HTML
    <div class="container">
      <p class="text">Первый параграф</p>
      <p class="text">Второй параграф</p>
    </div>
  

      
    // JavaScript
    let firstParagraph = document.querySelector(".text");
    console.log(firstParagraph.textContent); // "Первый параграф"

    Здесь querySelector находит первый элемент с классом text.

    1.3 querySelectorAll

    Если вам нужно найти все элементы, соответствующие селектору, используйте метод querySelectorAll. Он возвращает статическую коллекцию NodeList всех элементов, соответствующих указанному селектору.

    Пример:

      
    let allParagraphs = document.querySelectorAll(".text");
    allParagraphs.forEach(paragraph => console.log(paragraph.textContent));

    Этот код выводит текст всех параграфов с классом text.

  2. Манипуляция элементами DOM

    3. После того как вы нашли элементы на странице, вы можете изменять их, используя свойства и методы, предоставляемые DOM.

    2.1 Изменение текста элемента

    Свойство textContent позволяет изменить текст внутри элемента.

    Пример:

      
    let header = document.getElementById("main-header");
    header.textContent = "Новый заголовок";

    В результате текст внутри элемента с id "main-header" изменится на "Новый заголовок".

    2.2 Изменение HTML-содержимого

    Свойство innerHTML позволяет получить или изменить HTML-код внутри элемента.

    Пример:

      
    let container = document.querySelector(".container");
    container.innerHTML = "
    Новый параграф
    ";

    Этот код заменяет всё содержимое элемента с классом container на новый параграф.

    2.3 Изменение стилей

    Для изменения стилей элемента используется свойство style. Вы можете изменить любой стиль, определенный в CSS, через это свойство.

    Пример:

      
    let header = document.getElementById("main-header");
    header.style.color = "red";
    header.style.fontSize = "24px";

    Этот код изменяет цвет текста заголовка на красный и увеличивает размер шрифта до 24 пикселей.

    2.4 Изменение атрибутов

    Атрибуты HTML-элементов (например, src, href, class) можно изменять с помощью методов setAttribute, getAttribute и removeAttribute.

    Пример:

      
    let image = document.querySelector("img");
    image.setAttribute("src", "new-image.jpg");

    Этот код изменяет атрибут src изображения, чтобы оно отображало новый файл.

    2.5 Добавление и удаление классов

    Для работы с классами элементов используется свойство classList, которое предоставляет методы для добавления, удаления и переключения классов.

    Пример:

      
    let button = document.querySelector("button");
    button.classList.add("active");
    button.classList.remove("inactive");
    button.classList.toggle("highlight");

    Этот код добавляет класс active, удаляет класс inactive и переключает наличие класса highlight у кнопки.

  3. Работа с событиями

    4. Часто манипуляции с DOM происходят в ответ на события, такие как нажатие кнопки или изменение содержимого формы. Для обработки событий используется метод addEventListener.

    Пример:

      
    let button = document.querySelector("button");
    button.addEventListener("click", function() {
        alert("Кнопка нажата!");
    });

    Этот код показывает всплывающее сообщение, когда пользователь нажимает на кнопку.

Заключение

Поиск и манипуляция элементами DOM — это основа для создания интерактивных веб-приложений. Понимание того, как использовать методы getElementById, querySelector, а также изменять свойства, атрибуты и стили элементов, позволяет разработчикам динамически изменять контент страницы, создавая более отзывчивые и интерактивные интерфейсы.
