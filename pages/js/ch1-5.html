Операторы и выражения в JavaScript

Операторы в JavaScript — это специальные символы, которые выполняют операции над операндами (переменными или значениями) и возвращают результат. Выражение — это комбинация переменных, операторов и значений, которая вычисляется и возвращает результат.

Типы операторов в JavaScript

JavaScript поддерживает различные типы операторов:

  1. Арифметические операторы
  2. Операторы присваивания
  3. Строковые операторы
  4. Операторы сравнения
  5. Логические операторы
  6. Операторы работы с битами
  7. Операторы типа
  8. Специальные операторы

1. Арифметические операторы

Арифметические операторы используются для выполнения математических операций.

  • + (сложение): Складывает два числа.
    • let sum = 5 + 3; // 8
  • - (вычитание): Вычитает одно число из другого.

    • let difference = 10 - 4; // 6

  • * (умножение): Умножает два числа.

    • let product = 7 * 3; // 21

  • / (деление): Делит одно число на другое.

    • let quotient = 20 / 4; // 5

  • % (остаток от деления): Возвращает остаток от деления двух чисел.

    • let remainder = 10 % 3; // 1

  • ** (возведение в степень): Возводит число в степень.

    • let power = 2 ** 3; // 8

2. Операторы присваивания

Эти операторы присваивают значения переменным. Основной оператор — это =, но существуют и составные операторы присваивания, которые объединяют операцию и присваивание.

  • = Присваивает значение переменной.

    • let x = 10;

  • += Добавляет и присваивает.
    • let x = 5;
    x += 3; // x = x + 3; x теперь равен 8
  • -= Вычитает и присваивает.

    • let x = 5;
    x -= 2; // x теперь равен 3

  • *= Умножает и присваивает.

    • let x = 5;
    x *= 3; // x теперь равен 15

  • /= Делит и присваивает.

    • let x = 10;
    x /= 2; // x теперь равен 5

  • %= Делит и присваивает остаток.

    • let x = 10;
    x %= 3; // x теперь равен 1

3. Строковые операторы

Основной оператор для работы со строками — '+', который используется для конкатенации строк.

  • Конкатенация строк: Объединение двух или более строк.

    • let greeting = "Hello, " + "world!"; // "Hello, world!"

4. Операторы сравнения

Операторы сравнения используются для сравнения значений. Результатом сравнения является логическое значение 'true' или 'false'.

  • == Равно. Проверяет равенство двух значений, при этом не учитывается их тип.

    • 5 == "5"; // true

  • === Строгое равенство. Проверяет как значение, так и тип.

    • 5 === "5"; // false

  • != Не равно. Проверяет неравенство значений.

    • 5 != "5"; // false

  • !== Строгое неравенство. Проверяет неравенство значений и типов.

    • 5 !== "5"; // true

  • > Больше.

    • 10 > 5; // true

  • < Меньше.

    • 3 < 7; // true

  • >= Больше или равно.

    • 5 >= 5; // true

  • <= Меньше или равно.

    • 4 <= 8; // true

5. Логические операторы

Логические операторы используются для выполнения логических операций, которые возвращают значения 'true' или 'false'.

  • && (логическое И): Возвращает true, если оба операнда истинны.

    • let result = (5 > 3) && (8 > 5); // true

  • || (логическое ИЛИ): Возвращает true, если хотя бы один из операндов истинен.

    • let result = (5 > 3) || (8 < 5); // true

  • ! (логическое НЕ): Возвращает противоположное логическое значение.

    • let result = !(5 > 3); // false

6. Операторы работы с битами

Операторы работы с битами позволяют выполнять операции на уровне отдельных битов в числах.

  • & Побитовое И.
  • | Побитовое ИЛИ.
  • ^ Побитовое исключающее ИЛИ.
  • ~ Побитовое НЕ.
  • << Сдвиг влево.
  • >> Сдвиг вправо.

7. Операторы типа

Операторы типа позволяют определить тип переменной или преобразовать значения к определенному типу.

  • typeof Возвращает строку, которая обозначает тип переменной.

    • typeof "Hello"; // "string"
    typeof 42; // "number"
    typeof true; // "boolean"

  • instanceof Проверяет, является ли объект экземпляром определенного класса или конструктора.

    • let date = new Date();
    date instanceof Date; // true

8. Специальные операторы

  • Запятая (,): Позволяет вычислить несколько выражений, возвращая значение последнего.

    • let a = 1, b = 2, c = 3;

  • Тернарный оператор (?:): Упрощенная форма условного оператора if.

    • let age = 18;
    let isAdult = age >= 18 ? "Взрослый" : "Ребенок"; // "Взрослый"

  • Оператор delete: Удаляет свойство объекта.

    • let person = {name: "John", age: 30};
    delete person.age; // Удаляет свойство age
  • Оператор in: Проверяет, существует ли определенное свойство в объекте.

    • let person = {name: "John", age: 30};
    "age" in person; // true

  • Оператор void: Выполняет выражение, но всегда возвращает undefined.

    • void(0); // undefined
