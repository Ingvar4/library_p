    1. Введение в CSS

  • Что такое CSS и зачем он нужен.
  • Основные способы подключения CSS: inline, internal, external.
  • Основные синтаксические правила CSS.
  • Основные понятия: селекторы, свойства, значения.

    2. Базовые стили

  • Работа с текстом: font-size, font-family, font-weight, text-align, text-transform, line-height.
  • Цвета и фон: color, background-color, background-image, opacity.
  • Отступы и поля (margins и padding).
  • Бордеры и их свойства: border, border-radius.

    3. Селекторы в CSS

  • Простые селекторы: по тегам, классам, идентификаторам.
  • Комбинированные селекторы: потомки, дети, соседние элементы.
  • Псевдоклассы: :hover, :focus, :active, :nth-child(), :last-child().
  • Псевдоэлементы: ::before, ::after.
  • Атрибутные селекторы.

    4. Работа с блоками и макетом

  • Блочные и строчные элементы.
  • Свойства display: block, inline-block, none.
  • Модель блочной коробки (Box Model).
  • Введение в Flexbox: контейнер и элементы.
  • Введение в CSS Grid: работа с сетками.

    5. Позиционирование элементов

  • Статичное, относительное, абсолютное и фиксированное позиционирование.
  • Свойства top, right, bottom, left, z-index.
  • Современные макеты с помощью Flexbox и Grid.

    6. Адаптивный дизайн

  • Введение в адаптивную верстку.
  • Использование медиа-запросов (@media) для различных устройств.
  • Основы подхода "Mobile First".
  • Работа с резиновыми единицами: em, rem, vw, vh, %.

    7. Трансформации и анимации

  • Трансформации: transform, translate, rotate, scale, skew.
  • Переходы: transition, как задать временные интервалы и задержки.
  • Анимации: свойство animation, ключевые кадры @keyframes.
