Бордеры и их свойства: `border`, `border-radius`

В CSS свойства, связанные с рамками (бордерами), такие как border и border-radius, позволяют добавлять и стилизовать границы вокруг элементов. Эти свойства помогают создать акценты, отделить элементы или придать им более интересную форму.

  1. border (Граница)

    2. border — это свойство, которое задает рамку вокруг элемента. Оно может быть представлено как сокращенная запись для всех сторон границы (верхней, правой, нижней и левой), либо для каждой стороны можно задать рамку отдельно.

    Сокращенная запись:

    Свойство border можно задать через три значения:

    • Толщина границы (например, 1px, 2em, 0.5rem).
    • Стиль границы: solid (сплошная линия), dashed (пунктирная), dotted (точечная), double (двойная), groove, ridge, inset, outset, и другие.
    • Цвет границы (например, #ff0000, rgba(0, 0, 255, 0.5)).

    Пример:

      
    div {
      border: 2px solid #ff0000;
    }

    Этот код добавляет красную рамку вокруг блока <div> толщиной в 2 пикселя и со сплошным стилем.

    Задание границ для каждой стороны:

    CSS позволяет задавать разные границы для каждой стороны элемента с помощью свойств border-top, border-right, border-bottom, border-left:

      
    div {
      border-top: 3px dashed blue;
      border-right: 2px solid green;
      border-bottom: 4px double red;
      border-left: 1px dotted black;
    }

    Каждая сторона границы будет иметь свой стиль, толщину и цвет.

    Внутренние свойства border:

    • border-width: управляет толщиной границы.
    • border-style: задает стиль границы (например, solid, dashed, none).
    • border-color: определяет цвет границы. Может принимать любые цветовые значения, как и свойство color.

    Пример для управления отдельными свойствами:

      
    p {
      border-width: 5px;
      border-style: dotted;
      border-color: #00ff00;
    }

    Этот код задает зеленую точечную границу толщиной 5 пикселей вокруг абзаца.

  2. border-radius (Закругление углов)

    3. border-radius позволяет закруглять углы элементов, что делает блоки или другие элементы более «мягкими» и визуально привлекательными. Радиус можно задать как для всех углов элемента сразу, так и для каждого угла по отдельности.

    Сокращенная запись:

      
    div {
      border-radius: 10px;
    }

    Здесь задается радиус закругления в 10 пикселей для всех углов элемента. Это делает углы менее резкими и добавляет округлость.

    Задание радиуса для каждого угла:

    Порядок задания значений для углов такой же, как и у отступов (по часовой стрелке): верхний левый, верхний правый, нижний правый, нижний левый:

      
    div {
      border-radius: 10px 20px 30px 40px;
    }

    Это задаст разные радиусы для каждого угла: 10px для верхнего левого, 20px для верхнего правого, 30px для нижнего правого и 40px для нижнего левого.

    Радиус с процентами:

    Если использовать проценты, то радиус будет вычисляться относительно размеров элемента:

      
    div {
      border-radius: 50%;
    }

    Этот код сделает элемент с шириной и высотой одинакового значения полностью круглым (например, если это квадрат).

    Особенности:

    • Округление для картинок: Свойство border-radius полезно для создания круглых изображений:
      •   
    img {
      border-radius: 50%;
    }

      Если задать border-radius: 50% для изображения, оно станет кругом, если пропорции изображения квадратные.

    • Значение inherit: Оба свойства могут наследовать значения от родительского элемента с помощью inherit, что удобно для создания единообразного дизайна.

    Пример с комбинированием border и border-radius:

      
    div {
      width: 200px;
      height: 200px;
      background-color: lightblue;
      border: 5px solid blue;
      border-radius: 20px;
    }

    Этот стиль задает квадратный блок размером 200x200 пикселей с голубым фоном, синей границей толщиной 5px и закругленными углами радиусом 20px.

Заключение

  • border управляет границей элемента, её толщиной, стилем и цветом.
  • border-radius добавляет округлость углам элемента и может создавать различные визуальные эффекты, включая полные круги или овалы.

Оба свойства играют важную роль в создании более привлекательных и уникальных интерфейсов.
