Комбинированные селекторы в CSS

Комбинированные селекторы позволяют выбирать элементы на основе их отношений с другими элементами в документе.

  
    <style>
      /* Селектор потомков */
      .parent p {
          color: blue; /* Все элементы <p> внутри элемента с классом 'parent' */
      }

      /* Селектор детей */
      .parent > h2 {
          color: green; /* Только прямые потомки <h2> элемента с классом 'parent' */
      }

      /* Селектор соседей (смежный) */
      h2 + p {
          color: red; /* Первый <p>, который следует сразу за <h2> */
      }

      /* Селектор общего соседа (все соседние) */
      h2 ~ p {
          color: orange; /* Все <p>, которые следуют за <h2> на одном уровне */
      }
    </style>

1. Селектор потомков

Этот абзац находится внутри элемента с классом 'parent' и будет синим.

Этот абзац тоже потомок 'parent', поэтому тоже будет синим.

Этот абзац не является потомком элемента с классом 'parent', поэтому на него стиль не повлияет.

Пример использования:

.parent p {
        color: blue;
    }

2. Селектор детей

Этот заголовок — прямой потомок элемента с классом 'parent', и он будет зеленым.

Этот заголовок не является прямым потомком 'parent', поэтому на него стиль не повлияет.

Пример использования:

.parent > h2 {
        color: green;
    }

3. Селектор соседей (смежный)

Этот заголовок

Этот абзац идет сразу после заголовка <h2>, и он будет красным.

Пример использования:

h2 + p {
        color: red;
    }

4. Селектор общего соседа

Этот заголовок

Этот абзац — соседний элемент на одном уровне с заголовком <h2>, поэтому он будет оранжевым.

Этот абзац тоже находится на одном уровне и пойдет следом, так что он тоже станет оранжевым.

Пример использования:

h2 ~ p {
        color: orange;
    }

Подробное объяснение:

  1. Селектор потомков (descendant selector):
    • Этот селектор выбирает элементы, которые являются потомками указанного родительского элемента.
    • Пример: .parent p — выбирает все <p>, находящиеся внутри элемента с классом parent, даже если между ними есть другие элементы.
    • В примере выше все абзацы внутри элемента с классом parent будут синими.
  2. Селектор детей (child selector):
    • Этот селектор выбирает только прямых потомков указанного родителя.
    • Пример: .parent > h2 — выберет только те заголовки <h2>, которые являются прямыми детьми элемента с классом parent.
    • В примере первый заголовок <h2> будет зеленым, потому что он прямой потомок элемента с классом parent.
  3. Селектор соседей (adjacent sibling selector):
    • Этот селектор выбирает элемент, который непосредственно следует за указанным элементом на одном уровне иерархии.
    • Пример: h2 + p — выбирает первый абзац <p>, который идет сразу после заголовка <h2>.
    • В примере абзац, который идет сразу после <h2>, станет красным.
  4. Селектор общего соседа (general sibling selector):
    • Этот селектор выбирает все элементы, которые являются соседями указанного элемента на одном уровне.
    • Пример: h2 ~ p — выбирает все абзацы <p>, которые следуют за <h2> на том же уровне иерархии.
    • В примере все абзацы, идущие за <h2>, станут оранжевыми.
