Основные понятия: селекторы, свойства, значения.

  1. Селекторы в CSS

    2. Селекторы — это инструменты, которые позволяют выбирать HTML-элементы на странице для их дальнейшей стилизации. Именно благодаря селекторам мы можем сказать браузеру, какие элементы нужно изменить и какие стили к ним применить.

    Виды селекторов:

    1. Селекторы по тегу: Селекторы по тегу применяют стили ко всем элементам определенного типа на странице.
      2.   
    p {
      color: blue;
    }
    2. Селекторы по классу: Селекторы классов позволяют выбрать элементы с определенным классом. В HTML для элемента можно задать атрибут class, который можно использовать для селектора в CSS. Перед именем класса ставится точка (.).
      3.   
    .button {
      background-color: green;
    }

      Этот стиль будет применен ко всем элементам с классом button. Например:

        
    <button class="button">Нажми меня</button>
    3. Селекторы по идентификатору (id): Селекторы идентификаторов позволяют выбрать элемент по его уникальному id. В отличие от классов, идентификатор должен быть уникален на странице. Для выбора элемента по id в CSS используется символ решетки (#).
      4.   
    #header {
      background-color: yellow;
    }

      Этот стиль применится к элементу с идентификатором header. Например:

        
    <div id="header">Заголовок</div>
    4. Групповые селекторы: Чтобы применить один и тот же набор стилей к нескольким элементам, можно объединить селекторы через запятую.
      5.   
    h1, h2, p {
      color: black;
    }

      Этот стиль применит черный цвет ко всем заголовкам <h1>, <h2> и абзацам <p>.

    5. Комбинированные селекторы: Можно выбирать элементы, основываясь на их родительских элементах или комбинациях классов и тегов. Например:
      6.   
    div p {
      color: red;
    }

      Этот стиль будет применен только к элементам <p>, которые находятся внутри элемента <div>. А вот стиль ниже применится ко всем абзацам <p>, у которых есть класс highlight:

        
    p.highlight {
      color: red;
    }
    6. Псевдоклассы: Псевдоклассы позволяют выбирать элементы в определенных состояниях или на основе их положения среди других элементов. Например, :hover используется для изменения стилей элемента при наведении на него мыши:
      7.   
    a:hover {
      color: red;
    }

      Этот стиль изменит цвет ссылки на красный при наведении на нее.

    7. Псевдоэлементы: Псевдоэлементы позволяют стилизовать определенные части элемента, такие как первый символ или строка, или добавлять контент до или после элемента. Примеры:
      • ::before: добавляет контент перед элементом.
      • ::after: добавляет контент после элемента.
        
    p::before {
      content: "→ ";
      color: gray;
    }

      Этот стиль добавит стрелку перед каждым абзацем <p>.

  2. Свойства в CSS

    3. Свойства в CSS задают, какие именно стили будут применены к выбранным элементам. Свойства описывают конкретный аспект внешнего вида элемента: цвет, шрифт, отступы, размеры, фон и многое другое.

    Примеры свойств:

    1. color — задает цвет текста.
      2.   
    color: blue;
    2. background-color — устанавливает цвет фона элемента.
      3.   
    background-color: yellow;
    3. font-size — определяет размер текста.
      4.   
    font-size: 16px;
    4. margin — задает внешние отступы вокруг элемента.
      5.   
    margin: 10px;
    5. padding — определяет внутренние отступы между содержимым элемента и его границами.
      6.   
    padding: 15px;
    6. border — добавляет границу вокруг элемента.
      7.   
    border: 1px solid black;
    7. display — определяет, как элемент будет отображаться на странице.
      8.   
    display: block;
    8. position — управляет расположением элемента.
      9.   
    position: absolute;
    9. text-align — выравнивает текст внутри элемента по левому, правому краю или центру.
      10.   
    text-align: center;
  3. Значения в CSS

    4. Значения — это конкретные параметры, которые присваиваются свойствам. Каждое свойство CSS принимает определенные типы значений, такие как длины, цвета, ключевые слова и т. д.

    Основные типы значений:

    1. Цвета:
      • Цвета могут задаваться с помощью ключевых слов (red, blue), шестнадцатеричных кодов (#ff0000), RGB (rgb(255, 0, 0)) и других форматов.
      2.   
    color: red;
    background-color: #ff0000;
    2. Длины:
      • Длины указываются в абсолютных единицах (например, px, cm) или относительных (например, em, %). 
      3.   
    font-size: 16px;
    margin: 10%;
    3. Проценты:
      • Проценты используются для задания значений относительно родительского элемента или контейнера. 
      4.   
    width: 50%;
    4. Ключевые слова:
      • Некоторые свойства принимают специальные ключевые слова. Например, для свойства display существуют такие ключевые значения, как block, inline, none. 
      5.   
    display: block;
    5. Специальные значения:
      • CSS поддерживает значения вроде inherit, initial, auto:
        • inherit: наследует значение свойства от родительского элемента.
        • initial: сбрасывает свойство до значения по умолчанию.
        • auto: используется для автоматического расчета значения браузером.
      6.   
    width: auto;

Примеры использования свойств и значений:

  1. Стилизация текста:
    2.     
    p {
      color: blue;
      font-size: 18px;
      font-family: Arial, sans-serif;
    }

    Этот стиль применяет синий цвет текста, шрифт Arial и размер текста 18px ко всем абзацам.

  2. Работа с фоном и отступами:
    3.   
    div {
      background-color: lightgray;
      margin: 20px;
      padding: 10px;
    }

    Этот код задает серый фон для всех <div>, отступ в 20px вокруг блока и внутренний отступ (padding) в 10px.

Заключение

Селекторы, свойства и значения — это три основополагающие составляющие CSS. Селекторы позволяют выбирать элементы, свойства задают, какой аспект элемента будет изменен, а значения определяют конкретные параметры этих изменений. Понимание этих понятий поможет тебе грамотно и эффективно использовать CSS для стилизации веб-страниц.
