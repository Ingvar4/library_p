Трансформации: transform, translate, rotate, scale, skew.

Трансформации в CSS позволяют изменять вид элементов, применяя к ним различные геометрические операции. Они помогают создавать интересные и динамичные визуальные эффекты без использования JavaScript или сложных анимаций. Рассмотрим всё, что необходимо знать о трансформациях.

Свойство transform

  • Описание: transform — это основное свойство, с помощью которого можно применять различные трансформации к элементам, такие как перемещение, поворот, масштабирование и наклон.
  • Синтаксис:
    •   
    /*CSS*/

    transform: [функция_трансформации] [функция_трансформации] ...;

    Вы можете комбинировать несколько трансформаций, используя пробел для разделения.

  • Пример:
    •   
    /*CSS*/

    .box {
      transform: rotate(45deg) translateX(50px);
    }

Общие замечания:

  • Трансформации могут влиять на поток документа. Например, элементы, перемещённые с помощью translate, сохраняют своё исходное место в потоке.
  • Свойство transform поддерживает 2D и 3D трансформации.

1. translate()

  • Описание: Перемещает элемент по осям X и Y без влияния на другие элементы в потоке.
  • Синтаксис:
    •   
    /*CSS*/

    transform: translate(x, y);
    • x и y могут быть указаны в пикселях (px), процентах (%), em, и других единицах.
    • Если y не указан, он по умолчанию равен 0.
  • Примеры:
    •   
    /*CSS*/

    .move-right {
      transform: translateX(50px); /* Переместить вправо на 50px */
    }
    .move-down {
      transform: translateY(100px); /* Переместить вниз на 100px */
    }
    .move-diagonal {
      transform: translate(30px, 40px); /* Переместить по обеим осям */
    }

Особенности:

  • Трансформации не изменяют позиционирование в потоке, поэтому другие элементы не смещаются.

2. rotate()

  • Описание: Поворачивает элемент на заданный угол относительно центра элемента.
  • Синтаксис:
    •   
    /*CSS*/

    transform: rotate(angle);
    • angle — это угол поворота в градусах (deg), радианах (rad), градах (grad) или оборотах (turn).
  • Примеры:
    •   
    /*CSS*/

    .rotate-45 {
      transform: rotate(45deg); /* Повернуть на 45 градусов */
    }
    .rotate-negative {
      transform: rotate(-30deg); /* Повернуть против часовой стрелки */
    }

Советы:

  • Поворот происходит вокруг центра элемента. Чтобы изменить эту точку, можно использовать свойство transform-origin.
  • Положительные значения поворачивают элемент по часовой стрелке, отрицательные — против.

3. scale()

  • Описание: Масштабирует элемент по осям X и Y.
  • Синтаксис:
    •   
    /*CSS*/

    transform: scale(x, y);
    • x и y — это множители масштабирования. Значение 1 означает отсутствие изменения размера.
    • Если y не указан, он принимает значение x.
  • Примеры:
    •   
    /*CSS*/

    .scale-up {
      transform: scale(1.5); /* Увеличить размер в 1.5 раза */
    }
    .scale-down {
      transform: scale(0.5); /* Уменьшить размер вдвое */
    }
    .stretch {
      transform: scale(2, 0.5); /* Растянуть по X в 2 раза, сжать по Y вдвое */
    }

Замечания:

  • Масштабирование изменяет размеры элемента, но не его положение.
  • Использование значений меньше 1 уменьшает элемент, больше 1 — увеличивает.

4. skew()

  • Описание: Искажает элемент, наклоняя его по осям X и Y.
  • Синтаксис:
    •   
    /*CSS*/

    transform: skew(x-angle, y-angle);
    • x-angle и y-angle — углы наклона в градусах. Если y-angle не указан, он равен 0.
  • Примеры:
    •   
    /*CSS*/

    .skew-x {
      transform: skewX(30deg); /* Наклонить по оси X */
    }
    .skew-y {
      transform: skewY(20deg); /* Наклонить по оси Y */
    }
    .skew-both {
      transform: skew(15deg, 25deg); /* Наклонить по обеим осям */
    }

Особенности:

  • Наклон изменяет внешний вид элемента, но не его размеры или положение в потоке.
  • Искажение особенно полезно для создания динамичных и необычных дизайнов.

Свойство transform-origin

  • Описание: Определяет точку, относительно которой применяется трансформация.
  • Синтаксис:
    •   
    /*CSS*/

    transform-origin: x y;
    • x и y могут быть указаны в процентах, пикселях, или такими значениями, как left, right, top, bottom, center.
  • Пример:
    •   
    /*CSS*/

    .rotate-custom {
      transform-origin: top left; /* Повернуть относительно верхнего левого угла */
      transform: rotate(45deg);
    }

Использование:

  • Меняя transform-origin, можно добиваться разнообразных эффектов вращения и масштабирования.

Комбинирование трансформаций

  • Трансформации можно объединять для создания сложных эффектов:
    •   
    /*CSS*/

    .complex-transform {
      transform: translate(50px, 100px) rotate(30deg) scale(1.2);
    }
  • Порядок имеет значение: сначала выполняется translate, затем rotate, а потом scale.

Примеры использования:

  • Анимации наводки (hover-эффекты).
  • Создание сложных визуальных эффектов и анимаций.

3D-Трансформации

CSS позволяет создавать 3D-эффекты с помощью дополнительных функций и параметров. Рассмотрим основные:

1. perspective

  • Описание: Свойство perspective определяет, насколько элемент будет искажаться при трансформациях в 3D-пространстве. Чем меньше значение perspective, тем сильнее искажение.
  • Синтаксис:
    •   
    /*CSS*/

    perspective: value;
    • value — расстояние от зрителя до плоскости преобразования в пикселях.
  • Пример:
    •   
    /*CSS*/

    .perspective-container {
      perspective: 800px;
    }

Замечания:

  • perspective применяется к родительскому элементу, чтобы создать эффект глубины для всех его дочерних элементов.
  • Комбинируется с другими 3D-трансформациями, такими как rotateX, rotateY.

2. rotateX() и rotateY()

  • Описание: Эти функции позволяют вращать элемент вокруг осей X и Y.
  • Синтаксис:
    •   
    /*CSS*/

    transform: rotateX(angle);
    transform: rotateY(angle);
    • angle указывается в градусах.
  • Примеры:
    •   
    /*CSS*/

    .rotate-x {
      transform: rotateX(45deg); /* Вращение вокруг оси X */
    }
    .rotate-y {
      transform: rotateY(90deg); /* Вращение вокруг оси Y */
    }

Особенности:

  • Вместе с perspective создают реалистичные 3D-эффекты.
  • Используются для анимаций и создания трехмерного пространства.

3. translateZ() и scaleZ()

  • Описание: Эти функции перемещают и масштабируют элемент по оси Z, добавляя глубину.
  • Синтаксис:
    •   
    /*CSS*/

    transform: translateZ(value);
    transform: scaleZ(value);
    • value — величина перемещения или масштабирования.
  • Пример:
    •   
    /*CSS*/

    .move-closer {
      transform: translateZ(100px); /* Перемещение по оси Z */
    }

Замечания:

  • Полезны при создании элементов с глубиной в 3D-сценах.
  • Использование оси Z может потребовать дополнительных настроек perspective.

4. transform-style

  1. Описание: Определяет, как должны отображаться дочерние элементы в 3D-пространстве.
  2. Синтаксис:
    3.   
    /*CSS*/

    transform-style: flat | preserve-3d;
    • flat (по умолчанию): дочерние элементы не сохраняют свои 3D-преобразования.
    • preserve-3d: дочерние элементы сохраняют 3D-преобразования.
  3. Пример:
    4.   
    /*CSS*/

    .parent {
      transform-style: preserve-3d;
    }

Применение:

  • preserve-3d необходим, если вы хотите, чтобы дочерние элементы оставались в 3D-пространстве.

5. backface-visibility

  • Описание: Определяет, видна ли задняя сторона элемента, когда он поворачивается в 3D-пространстве.
  • Синтаксис:
    •   
    /*CSS*/

    backface-visibility: visible | hidden;
    • visible (по умолчанию): задняя сторона видима.
    • hidden: задняя сторона скрыта.
  • Пример:
    •   
    /*CSS*/

    .flipping-card {
      backface-visibility: hidden;
    }

Использование:

  • Полезно при создании эффектов переворота, таких как карточки.

Продвинутые техники и советы

  1. Комбинирование трансформаций:
    • Вы можете объединять 2D и 3D трансформации, чтобы создавать сложные анимации и эффекты.
    • Пример:
      •   
    /*CSS*/

    .complex-effect {
      transform: rotateX(45deg) translateZ(100px) scale(1.5);
    }
  2. Анимация с @keyframes:
    • Трансформации отлично сочетаются с анимациями.
    • Пример:
      •   
    /*CSS*/

    @keyframes spin {
      from {
        transform: rotateY(0deg);
      }
      to {
        transform: rotateY(360deg);
      }
    }
    
    .spinning-box {
      animation: spin 5s infinite linear;
    }

Практические советы:

  • Используйте will-change: transform, чтобы заранее сообщить браузеру о грядущих трансформациях и оптимизировать производительность.
  • Помните, что чрезмерное использование 3D-трансформаций может замедлить рендеринг, особенно на мобильных устройствах.
