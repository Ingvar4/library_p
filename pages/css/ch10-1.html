CSS Grid для сложных макетов.

CSS Grid — это мощный инструмент для создания сложных и адаптивных макетов, подходящий для многоуровневых и гибких компоновок элементов на странице. Он позволяет управлять расположением как основных, так и вложенных элементов в двухмерной сетке — по осям X и Y. В отличие от Flexbox, предназначенного больше для однонаправленных макетов, Grid позволяет планировать расположение элементов как в строках, так и в колонках, что делает его идеальным для компоновки целых страниц или крупных контейнеров.

Основные концепции CSS Grid

1. Grid-контейнер и Grid-элементы

  • Grid-контейнер — это элемент, которому присваивается свойство display: grid;. Внутри контейнера находится сетка, и все дочерние элементы этого контейнера становятся Grid-элементами.
  • Grid-элементы — это дочерние элементы Grid-контейнера, которые располагаются на сетке.
  
    /*CSS*/

    .container {
      display: grid;
    }

2. Основные свойства Grid-контейнера

2.1 grid-template-columns и grid-template-rows

Эти свойства задают количество и размер колонок и строк в сетке.

  
    /*CSS*/

    .container {
      display: grid;
      grid-template-columns: 1fr 2fr 1fr; /* три колонки с разной шириной */
      grid-template-rows: 100px 200px;   /* две строки разной высоты */
    }
  • 1fr — единица fr представляет собой долю свободного пространства в контейнере.
  • Также можно задавать фиксированные значения (px, em, %) и автоматическое (auto) значение.

2.2 gap, column-gap, row-gap

Свойство gap задаёт отступы между ячейками сетки, а column-gap и row-gap отдельно управляют расстоянием между колонками и строками.

  
    /*CSS*/

    .container {
      display: grid;
      grid-template-columns: 1fr 1fr;
      grid-template-rows: 200px 200px;
      gap: 10px; /* общий отступ между ячейками */
    }

2.3 grid-template-areas

Это свойство определяет именованные области сетки, что позволяет создавать макеты с определёнными областями и упрощает управление стилями.

  
    /*CSS*/

    .container {
      display: grid;
      grid-template-columns: 1fr 2fr;
      grid-template-rows: 100px 100px;
      grid-template-areas: 
        "header header"
        "sidebar content";
    }
    
    .header {
      grid-area: header;
    }
    
    .sidebar {
      grid-area: sidebar;
    }
    
    .content {
      grid-area: content;
    }

3. Расположение Grid-элементов

3.1 grid-column и grid-row

Эти свойства задают начальную и конечную позицию элемента по колонкам и строкам, управляя его положением и размером.

  
    /*CSS*/

    .item {
      grid-column: 1 / 3; /* занимает первую и вторую колонку */
      grid-row: 1 / 2;    /* занимает первую строку */
    }

3.2 grid-area

Можно задать расположение элемента с помощью одной строки с grid-area, комбинируя его с grid-template-areas или указывая начальные и конечные строки и колонки.

  
    /*CSS*/

    .item {
      grid-area: 1 / 2 / 3 / 4; /* занимает строки 1-3 и колонки 2-4 */
    }

4. Автоматическая расстановка элементов

4.1 grid-auto-rows и grid-auto-columns

Задают высоту строк и ширину колонок, создаваемых автоматически при добавлении новых элементов.

  
    /*CSS*/

    .container {
      display: grid;
      grid-auto-rows: 100px;
      grid-auto-columns: 1fr;
    }

4.2 grid-auto-flow

Определяет, как будут располагаться автоматически добавленные элементы — по строкам или колонкам.

  
    /*CSS*/

    .container {
      display: grid;
      grid-auto-flow: row; /* элементы заполняют строки, затем переходят к новой */
    }

5. Продвинутые шаблоны макетов

5.1 Responsive Grid

Для адаптивной сетки часто используются minmax() и repeat().

  • minmax(min, max) — задает диапазон размера ячейки. Например, grid-template-columns: repeat(3, minmax(100px, 1fr)); создаёт три колонки, где минимальная ширина — 100px, а максимальная — 1fr.
    •   
    /*CSS*/

    .container {
      display: grid;
      grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    }
  • auto-fill и auto-fit — полезны для создания сетки, которая адаптируется к ширине контейнера.
    •   
    /*CSS*/

    .container {
      display: grid;
      grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
    }

5.2 Комплексное позиционирование

  • justify-items и align-items — для выравнивания всех элементов по горизонтали и вертикали соответственно.
  • justify-self и align-self — выравнивание конкретного элемента в сетке.
    •   
    /*CSS*/

    .container {
      display: grid;
      justify-items: center;
      align-items: center;
    }
    
    .item {
      justify-self: end;
      align-self: start;
    }

5.3 Nested Grids

Grid поддерживает вложенные сетки для создания комплексных макетов, когда внутри одной ячейки создается новый Grid-контейнер.

  
    /*CSS*/

    .parent {
      display: grid;
      grid-template-columns: 1fr 2fr;
    }
    
    .child {
      display: grid;
      grid-template-columns: repeat(3, 1fr);
    }

Заключение

CSS Grid — это мощный инструмент для создания адаптивных и сложных макетов. В комбинации с медиазапросами, Grid позволяет реализовать продвинутые макеты, которые автоматически подстраиваются под размеры экрана.
