Использование CSS-переменных (--var и var()).

CSS-переменные (кастомные свойства) и препроцессоры играют ключевую роль в оптимизации и улучшении гибкости стилей, позволяя проще управлять изменениями и облегчать сложные проекты. Рассмотрим детально CSS-переменные, их использование и разницу между переменными CSS и переменными в CSS-препроцессорах (например, Sass или Less), а также разберем, как их комбинировать для более мощного и гибкого подхода к стилям.

1. Основы CSS-переменных

CSS-переменные, также известные как кастомные свойства, позволяют задавать значения, которые могут быть использованы многократно и динамически изменены, повышая гибкость стилей и упрощая их настройку.

Синтаксис CSS-переменных:

  • Переменные в CSS задаются через префикс -- в селекторах.
  • Для использования переменной применяется функция var().

Пример:

  
    /*CSS*/

    :root {
      --primary-color: #3498db;
      --font-size: 16px;
    }
    
    body {
      color: var(--primary-color);
      font-size: var(--font-size);
    }

Особенности:

  • Переменные, определенные в селекторе :root, становятся глобальными и доступны по всему документу.
  • Можно переопределять переменные в любом контейнере, что позволяет изменять значения в определенных зонах страницы.

2. Динамическое использование CSS-переменных

CSS-переменные позволяют динамически изменять значения стилей в зависимости от условий и взаимодействий пользователя, например, через JavaScript или медиа-запросы.

Изменение CSS-переменных через JavaScript

Переменные могут обновляться динамически, что особенно полезно для создания светлой и темной тем или настройки предпочтений пользователя.

  
    //JS

    document.documentElement.style.setProperty('--primary-color', '#2ecc71');

Использование в медиа-запросах

CSS-переменные можно переопределять в медиа-запросах, что упрощает адаптивность дизайна.

  
    /*CSS*/

    :root {
      --font-size: 16px;
    }
    
    @media (max-width: 768px) {
      :root {
        --font-size: 14px;
      }
    }
    
    body {
      font-size: var(--font-size);
    }

3. Значения по умолчанию и вложенность

С помощью var() можно задавать значения по умолчанию, что полезно при отсутствии необходимой переменной.

  
    /*CSS*/

    body {
      color: var(--text-color, #333); /* #333 будет использован, если --text-color не задан */
    }

4. Переменные в CSS и препроцессорах (Sass, Less)

До появления CSS-переменных для многократного использования значений применялись переменные в препроцессорах, таких как Sass и Less.

Отличия:

  1. CSS-переменные: динамические и работают на уровне DOM, что позволяет изменять их значения через JavaScript и применять к отдельным элементам.
  2. Переменные препроцессоров: компилируются на этапе сборки CSS и не меняются в браузере.

Пример Sass-переменных:

  
    /*SCSS*/

    $primary-color: #3498db;

    body {
      color: $primary-color;
    }

5. Комбинирование CSS-переменных и переменных препроцессоров

Использование CSS-переменных и препроцессоров вместе может сделать проект более мощным и гибким. Обычно препроцессорные переменные задают глобальные настройки (например, базовые цвета и шрифты), а CSS-переменные применяют для динамических значений, например, тем или адаптивных стилей.

Пример использования Sass и CSS-переменных вместе:

  
    /*SCSS*/

    $base-color: #3498db;

    :root {
      --button-color: $base-color;
    }

    button {
      background-color: var(--button-color);
    }

6. Углубление: CSS-функции с CSS-переменными

CSS-переменные отлично сочетаются с функциями CSS, такими как calc(), rgb(), hsl(), позволяя создавать сложные вычисления и динамические стили.

Пример использования calc() с CSS-переменными:

  
    /*CSS*/

    :root {
      --padding: 10px;
    }
    
    .container {
      padding: calc(var(--padding) * 2); /* Увеличение отступов в 2 раза */
    }

Пример с hsl():

  
    /*CSS*/

    :root {
      --main-hue: 200;
    }
    
    body {
      color: hsl(var(--main-hue), 50%, 50%);
    }

7. Вариативные темы с CSS-переменными

С помощью CSS-переменных можно создать вариативные темы, например, для переключения между светлым и темным режимами.

  
    /*CSS*/

    :root {
      --background-color: #fff;
      --text-color: #000;
    }
    
    [data-theme="dark"] {
      --background-color: #000;
      --text-color: #fff;
    }
    
    body {
      background-color: var(--background-color);
      color: var(--text-color);
    }

8. Наследование и каскадирование CSS-переменных

CSS-переменные подчиняются правилам наследования и каскадирования, что может создавать гибкие структуры, но также приводит к возможным конфликтам.

Наследование

Переменные, объявленные в :root, доступны всем элементам, а переменные, заданные в конкретном контейнере, наследуются дочерними элементами.

Приоритет

Как и другие CSS-правила, переменные следуют приоритетам специфичности и могут быть переопределены через !important.

Пример:

  
    /*CSS*/

    :root {
      --font-size: 16px;
    }
    
    .container {
      --font-size: 20px;
    }
    
    .container p {
      font-size: var(--font-size); /* будет 20px */
    }

9. CSS Grid и Flexbox с CSS-переменными

CSS-переменные отлично работают с Grid и Flexbox, упрощая управление размером колонок и строк или отступами между элементами.

Пример с Grid:

  
    /*CSS*/

    :root {
      --column-width: 1fr;
    }
    
    .grid {
      display: grid;
      grid-template-columns: repeat(3, var(--column-width));
    }

10. Итог

CSS-переменные обеспечивают удобный и гибкий способ управления стилями, особенно для больших проектов и адаптивного дизайна. Они позволяют динамически изменять значения через JavaScript, использовать с медиа-запросами и управлять темами. В сочетании с препроцессорами CSS-переменные дают возможность более эффективно организовывать код и добиваться сложных эффектов с минимальными затратами.
