Современные макеты с помощью Flexbox и Grid.

Основные различия между Flexbox и Grid

  1. Flexbox предназначен для создания однорядных или одно-колоночных макетов. Он работает лучше при выравнивании элементов в одном направлении — либо по строкам, либо по столбцам.
  2. Grid — это система двухмерной разметки, которая позволяет размещать элементы по строкам и столбцам одновременно. Grid подходит для более сложных макетов с несколькими слоями и разделами.

Когда использовать Flexbox и Grid

  • Используйте Flexbox, если:
    • Вам нужно выравнивание элементов в одной строке или столбце.
    • Вы работаете с простыми интерфейсами, где элементы должны распределяться равномерно.
  • Используйте Grid, если:
    • Вам нужно создать сложные и гибкие макеты, включая панели с несколькими строками и колонками.
    • Вам требуется структурировать пространство сайта с более контролируемым расположением.

Подробности по Flexbox

Основные свойства Flexbox

  1. display: flex; — создаёт гибкий контейнер, все прямые дети которого становятся flex-элементами.
  2. flex-direction — определяет основную ось для распределения элементов.
    • row (по умолчанию): элементы располагаются в строку слева направо.
    • column: элементы располагаются в столбец сверху вниз.
    • row-reverse / column-reverse: меняют порядок на противоположный.
  3. justify-content — выравнивает элементы вдоль основной оси.
    • flex-start, flex-end, center, space-between, space-around, space-evenly.
  4. align-items — выравнивает элементы вдоль поперечной оси.
    • stretch (по умолчанию), flex-start, flex-end, center, baseline.
  5. align-self — позволяет отдельному элементу переопределить свойство align-items.
  6. flex-wrap — определяет, как элементы переносятся на новые строки.
    • nowrap (по умолчанию): все элементы в одной строке.
    • wrap: элементы переносятся на новые строки.
  7. flex-grow, flex-shrink, flex-basis — определяют, как элементы изменяются в размерах.

Примеры распространённых макетов с Flexbox

  1. Центрирование блока:
    2.   
    /*CSS*/

    .container {
      display: flex;
      justify-content: center;
      align-items: center;
    }
  2. Гибкие карточки:
    3.   
    /*CSS*/

    .card-container {
      display: flex;
      flex-wrap: wrap;
      gap: 20px;
    }
    .card {
      flex: 1 1 calc(33.33% - 20px); /* Гибкая ширина */
    }

Подробности по Grid

Основные свойства Grid

  1. display: grid; — создаёт grid-контейнер.
  2. grid-template-columns / grid-template-rows — задают размеры колонок и строк.
    • Например, grid-template-columns: repeat(3, 1fr); создаёт три равные колонки.
  3. grid-column / grid-row — определяют, как элемент растягивается через сетку.
    • grid-column: 1 / 3; растягивает элемент с первой до третьей колонки.
  4. gap, row-gap, column-gap — задают отступы между строками и колонками.
  5. grid-area — задаёт именованную область для использования в сетке.
  6. justify-items / align-items — выравнивают элементы в пределах ячеек.
  7. justify-content / align-content — выравнивают всю сетку внутри контейнера.

Сложные макеты с Grid

  1. Создание карточного макета:
    2.   
    /*CSS*/

    .grid-container {
      display: grid;
      grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
      gap: 20px;
    }
    .card {
      background: #f0f0f0;
      padding: 20px;
    }
  2. Макет с областями (named grid areas):
    3.   
    /*CSS*/

    .grid-layout {
      display: grid;
      grid-template-areas:
        "header header"
        "sidebar content"
        "footer footer";
      grid-template-columns: 200px 1fr;
      grid-template-rows: auto 1fr auto;
    }
    .header { grid-area: header; }
    .sidebar { grid-area: sidebar; }
    .content { grid-area: content; }
    .footer { grid-area: footer; }

Комбинация Flexbox и Grid

В реальных проектах вы можете использовать Grid для общей структуры страницы, а Flexbox для выравнивания отдельных элементов внутри ячеек сетки. Это сочетание создаёт гибкие и легко адаптируемые макеты.

Заключение

  • Оба метода мощны и имеют свои сильные стороны.
  • Используйте Flexbox для линейного выравнивания и небольших компонентов, таких как навигация или кнопки.
  • Используйте Grid для создания основного макета страницы с несколькими уровнями и сложными структурами.

Flexbox и Grid меняют подход к вёрстке, позволяя быстро и эффективно создавать адаптивные и упорядоченные макеты.
