Углубленное использование псевдоклассов и псевдоэлементов.

Псевдоклассы обозначаются двоеточием : и применяются для стилизации элементов в зависимости от их состояния, расположения или взаимодействия с пользователем.

1. Структурные псевдоклассы

Структурные псевдоклассы позволяют выбрать элемент на основе его положения в DOM или его отношений с другими элементами.

  • :first-child и :last-child — выбирают первый и последний дочерний элемент внутри родителя.
    •   
    /*CSS*/

    .container p:first-child {
      color: blue;
    }
  • :nth-child(n) и :nth-of-type(n) — позволяют выбрать элемент по порядковому номеру. Можно использовать формулы, например 2n (каждый второй) или 2n+1 (каждый нечетный).
    •   
    /*CSS*/

    li:nth-child(2n) {
      background-color: #f0f0f0;
    }
  • :only-child — выбирает элемент, который является единственным дочерним элементом родителя.
    •   
    /*CSS*/

    .container p:only-child {
      font-weight: bold;
    }
  • :not(selector) — исключает элементы, соответствующие указанному селектору, полезно для обратного выбора.
    •   
    /*CSS*/

    a:not(.active) {
      color: gray;
    }

2. Псевдоклассы состояния элемента

Эти псевдоклассы используются для выбора элементов в зависимости от их состояния, особенно полезны для интерактивных компонентов.

  • :hover — применяет стиль при наведении курсора на элемент.
  • :focus — активируется, когда элемент получает фокус, что полезно для полей форм и ссылок.
  • :active — применяется к элементу в момент нажатия.
    •   
    /*CSS*/

    button:active {
      background-color: #ddd;
    }
  • :disabled и :enabled — позволяют стилизовать элементы форм в зависимости от их состояния активности.
    •   
    /*CSS*/

    input:disabled {
      background-color: #f0f0f0;
    }

3. Псевдоклассы для ссылок

  • :link и :visited — стилизуют ссылки, которые ещё не были посещены или уже были посещены.
  
    /*CSS*/

    a:visited {
      color: purple;
    }

4. Псевдоклассы UI элементов

CSS также поддерживает UI-псевдоклассы для интерактивных элементов:

  • :checked — применяется к отмеченным элементам (input[type="checkbox"], input[type="radio"]).
  • :valid и :invalid — позволяют стилизовать поля форм в зависимости от их корректности.
    •   
    /*CSS*/

    input:valid {
      border-color: green;
    }

5. Псевдокласс :root

:root выбирает корневой элемент документа (html) и используется часто для CSS-переменных, чтобы задать глобальные стили.

  
    /*CSS*/

    :root {
      --main-bg-color: #3498db;
    }

Псевдоэлементы в CSS

Псевдоэлементы позволяют создавать и стилизовать части элемента или добавлять контент, который существует только в CSS, без дополнительных тегов HTML.

1. Основные псевдоэлементы

Псевдоэлементы обозначаются двойным двоеточием ::.

  • ::before и ::after — вставляют контент до или после контента элемента. Обычно используются для декоративных целей или добавления иконок.
    •   
    /*CSS*/

    .button::before {
      content: "→ ";
    }
  • ::first-line — стилизует первую строку текста элемента.
    •   
    /*CSS*/

    p::first-line {
      font-weight: bold;
    }
  • ::first-letter — применяется к первой букве текста, часто используется для стилизации заглавных букв в абзацах.
    •   
    /*CSS*/

    p::first-letter {
      font-size: 2em;
    }
  • ::selection — изменяет стиль выделенного пользователем текста, что может улучшить визуальную привлекательность.
    •   
    /*CSS*/

    ::selection {
      background-color: #ffb7b7;
    }

2. Комбинирование псевдоклассов и псевдоэлементов

CSS позволяет комбинировать псевдоклассы и псевдоэлементы, чтобы достичь сложных эффектов. Например, можно создать анимацию только при наведении:

  
    /*CSS*/

    .button:hover::after {
      content: "Hovered!";
      color: red;
    }

Практическое применение и примеры

Использование псевдоклассов для улучшения UI

  1. Декоративные эффекты с ::before и ::after: часто применяются для создания декоративных элементов, таких как стрелки, иконки, кнопки, разделители.
  2. Анимации с псевдоклассами: сочетание :hover, :focus, и :active с CSS-анимацией и трансформацией помогает создать интерактивные эффекты, улучшая UX.
  3. Условное отображение контента: например, :empty помогает скрыть элементы без содержимого, чтобы избежать пустого пространства на странице.

Эти возможности псевдоклассов и псевдоэлементов делают CSS мощным инструментом для создания сложных и интерактивных интерфейсов. Их комбинирование с медиа-запросами и анимацией CSS даёт контроль над визуальным отображением и взаимодействием на каждом уровне.
