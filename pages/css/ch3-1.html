Простые селекторы в CSS

Селекторы в CSS — это способ выбрать элементы HTML для стилизации. Простые селекторы — это основа, которую должен знать каждый, кто начинает изучать CSS.

1. Селектор по тегу

Селекторы по тегам выбирают все элементы с определенным тегом на странице. Например, в данном случае все теги <p> будут стилизованы синим цветом текста:

Пример использования:

p {
        color: #007bff;
        font-size: 16px;
    }

Этот стиль применится ко всем абзацам на странице, включая этот абзац.

2. Селектор по классу

Классы позволяют применять стили к нескольким элементам. Чтобы задать стиль для класса, в CSS перед названием класса используется точка.

Этот абзац выделен желтым фоном и жирным шрифтом с помощью класса highlight.

Пример использования:

.highlight {
        background-color: #ffff99;
        font-weight: bold;
    }

3. Селектор по идентификатору

Идентификаторы (ID) уникальны для каждого элемента и должны использоваться для того, чтобы выделить один конкретный элемент. В CSS селектор идентификатора начинается с символа решетки #.

Пример использования:

#main-title {
        font-size: 24px;
        color: #ff5722;
        text-align: center;
    }

Этот стиль применен к заголовку <h1 id="main-title"> в начале страницы.
