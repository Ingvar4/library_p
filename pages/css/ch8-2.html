Стилизация типичных элементов веб-страницы (шапка, меню, контент, подвал).

Создание стилизованного и структурированного макета веб-страницы — одна из ключевых задач при верстке. Мы разберем стилизацию основных компонентов веб-страницы: шапки (header), меню навигации (navigation), основного контента (content) и подвала (footer), включая продвинутые техники, обеспечивающие адаптивность и гибкость макета.

Шаблон страницы: структура и стилизация

1. Основная HTML-структура

Для начала создадим базовую HTML-структуру, которая будет содержать все основные элементы страницы.

  
    <!-- HTML -->

    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Стилизация веб-страницы</title>
      <link rel="stylesheet" href="styles.css">
    </head>
    <body>
      <header class="header">Заголовок страницы</header>
      <nav class="nav">Навигация</nav>
      <div class="content-wrapper">
        <aside class="sidebar">Боковая панель</aside>
        <main class="main">Основной контент</main>
      </div>
      <footer class="footer">Подвал</footer>
    </body>
    </html>

2. Базовая стилизация (CSS Reset)

Для консистентного отображения на всех устройствах и в браузерах, начнем с CSS Reset.

  
    /*CSS*/

    * {
      margin: 0;
      padding: 0;
      box-sizing: border-box;
    }

3. Основные стили для body и страницы

Установим базовые стили для фона страницы, шрифта и размера текста.

  
    /*CSS*/

    body {
      font-family: Arial, sans-serif;
      line-height: 1.6;
      color: #333;
      background-color: #f9f9f9;
      display: flex;
      flex-direction: column;
      min-height: 100vh;
    }

Стилизация Шапки (Header)

Шапка — это верхний блок, обычно содержащий логотип и основное меню. Для header добавим фоновый цвет, отступы и центровку текста.

  
    /*CSS*/

    .header {
      background-color: #4CAF50;
      color: #fff;
      padding: 1rem;
      text-align: center;
      font-size: 1.5rem;
    }

Стилизация Навигационного Меню (Navigation)

Навигационное меню часто располагается сразу под шапкой. Оно может быть фиксированным или гибким в зависимости от макета.

HTML-структура меню

  
    <!-- HTML -->

    <nav class="nav">
      <ul class="nav-list">
        <li><a href="#">Главная</a></li>
        <li><a href="#">О нас</a></li>
        <li><a href="#">Услуги</a></li>
        <li><a href="#">Контакты</a></li>
      </ul>
    </nav>

Стилизация меню

  
    /*CSS*/

    .nav {
      background-color: #333;
    }
    
    .nav-list {
      display: flex;
      justify-content: center;
      list-style: none;
    }
    
    .nav-list li {
      margin: 0 1rem;
    }
    
    .nav-list a {
      color: #fff;
      text-decoration: none;
      font-weight: bold;
    }
    
    .nav-list a:hover {
      color: #4CAF50;
    }

Мобильная адаптация меню

Для адаптивности добавим медиа-запросы.

  
    /*CSS*/

    @media (max-width: 768px) {
      .nav-list {
        flex-direction: column;
        align-items: center;
      }
      .nav-list li {
        margin: 0.5rem 0;
      }
    }

Стилизация Основного Контента и Боковой Панели

Основная часть контента может содержать как текст, так и медиа, таблицы и другие элементы. Боковая панель (sidebar) обычно используется для ссылок, виджетов или других вспомогательных блоков.

Структура и CSS-код для контейнера

Flexbox отлично подходит для построения макета с боковой панелью.

  
    /*CSS*/

    .content-wrapper {
      display: flex;
      flex: 1;
      padding: 1rem;
    }
    
    .sidebar {
      flex: 1;
      background-color: #e4e4e4;
      padding: 1rem;
      margin-right: 1rem;
      border-radius: 8px;
    }
    
    .main {
      flex: 3;
      background-color: #fff;
      padding: 1rem;
      border-radius: 8px;
    }

Адаптация для мобильных устройств

На мобильных устройствах элементы могут быть выстроены вертикально.

  
    /*CSS*/

    @media (max-width: 768px) {
      .content-wrapper {
        flex-direction: column;
      }
      .sidebar {
        margin-right: 0;
        margin-bottom: 1rem;
      }
    }

Стилизация Подвала (Footer)

Подвал страницы обычно используется для контактной информации, социальных ссылок или вспомогательных меню. Его можно стилизовать так, чтобы он был простым и отличался от остального макета.

  
    /*CSS*/

    .footer {
      background-color: #333;
      color: #fff;
      padding: 1rem;
      text-align: center;
      font-size: 0.9rem;
    }

Дополнительные элементы и анимации

Добавление эффектов для интерактивности

С помощью CSS анимаций можно добавить плавные эффекты для улучшения пользовательского опыта.

  
    /*CSS*/

    .nav-list a {
      color: #fff;
      text-decoration: none;
      font-weight: bold;
      transition: color 0.3s;
    }
    
    .nav-list a:hover {
      color: #4CAF50;
    }

Анимация кнопок

Для привлечения внимания к кнопкам добавим анимацию scale при наведении.

  
    /*CSS*/

    .button {
      background-color: #4CAF50;
      color: #fff;
      padding: 0.5rem 1rem;
      border: none;
      border-radius: 4px;
      cursor: pointer;
      transition: transform 0.2s;
    }
    
    .button:hover {
      transform: scale(1.05);
    }

Завершающий код CSS для мобильной адаптивности и интерактивности

  
    /*CSS*/

    @media (max-width: 768px) {
      .nav-list {
        flex-direction: column;
        align-items: center;
      }
    
      .content-wrapper {
        flex-direction: column;
      }
    
      .sidebar {
        margin: 0 0 1rem 0;
      }
    }
    
    a {
      color: #4CAF50;
      text-decoration: none;
      transition: color 0.3s;
    }
    
    a:hover {
      color: #333;
    }

Итог

Таким образом, с помощью комбинации Flexbox, CSS-свойств и медиа-запросов мы можем создать адаптивный макет для основных частей веб-страницы, который будет одинаково хорошо смотреться на любых устройствах.
