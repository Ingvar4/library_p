Введение в адаптивную верстку.

Адаптивная верстка (responsive web design) — это метод создания сайтов, которые корректно отображаются и работают на всех устройствах, независимо от размера экрана и разрешения. Суть адаптивной верстки заключается в том, чтобы сайт автоматически адаптировался под различные устройства, такие как настольные компьютеры, планшеты, смартфоны, а также устройства с нетипичными экранами.

Основные концепции адаптивной верстки

  1. Мета-тег Viewport
    • Определяет, как сайт масштабируется и отображается на устройствах с разным размером экрана.
      •   
    <!-- HTML -->

    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    • width=device-width: ширина страницы будет равна ширине устройства.
    • initial-scale=1.0: начальный масштаб равен 100%.
  2. Медиа-запросы (Media Queries)
    • Позволяют применять CSS-стили в зависимости от характеристик устройства, таких как ширина экрана, высота, ориентация и т.д.
      •   
    /*CSS*/

    @media (max-width: 768px) {
      body {
        font-size: 14px;
      }
    }
    • Примеры популярных брейкпоинтов:
      • max-width: 1200px (для больших экранов, ноутбуков)
      • max-width: 992px (для планшетов в ландшафтной ориентации)
      • max-width: 768px (для планшетов в портретной ориентации)
      • max-width: 576px (для смартфонов)
  3. Гибкие размеры (Fluid Grids and Layouts)
    • Использование процентов и единиц измерения, таких как em, rem, vh, vw, вместо фиксированных значений.
    • Например, задавая ширину контейнера:
      •   
    /*CSS*/

    .container {
      width: 100%; /* занимает всю доступную ширину */
    }
  4. Гибкие изображения и медиа
    • Изображения, видео и другие медиа должны изменять свои размеры в зависимости от экрана.
      •   
    /*CSS*/

    img {
      max-width: 100%;
      height: auto; /* сохраняет соотношение сторон */
    }

Подходы к адаптивной верстке

  1. Мобильный в первую очередь (Mobile-First Design)
    • Начинайте верстку с минимального размера экрана и добавляйте стили для больших экранов с помощью медиа-запросов.
    • Пример:
      •   
    /*CSS*/

    body {
      font-size: 16px; /* базовые стили для мобильных устройств */
    }
    
    @media (min-width: 768px) {
      body {
        font-size: 18px; /* для планшетов и больших экранов */
      }
    }
  2. Десктоп в первую очередь (Desktop-First Design)
    • Начинайте верстку с большего размера экрана и адаптируйте стили для меньших экранов.
    • Пример:
      •   
    /*CSS*/

    body {
      font-size: 18px; /* базовые стили для больших экранов */
    }
    
    @media (max-width: 768px) {
      body {
        font-size: 16px; /* уменьшение для мобильных устройств */
      }
    }
  3. Гибкие сетки (Flexible Grids)
    • Flexbox и Grid Layout часто используются для создания адаптивных макетов.
    • Flexbox:
      •   
    /*CSS*/

    .container {
      display: flex;
      flex-wrap: wrap;
    }
    
    .item {
      flex: 1 1 100%; /* элемент занимает всю ширину */
    }
    
    @media (min-width: 768px) {
      .item {
        flex: 1 1 50%; /* два элемента в строке на больших экранах */
      }
    }
    • Grid:
      •   
    /*CSS*/

    .grid-container {
      display: grid;
      grid-template-columns: 1fr;
      gap: 20px;
    }
    
    @media (min-width: 768px) {
      .grid-container {
        grid-template-columns: 1fr 1fr;
      }
    }

Адаптивная типографика

  • Использование гибких единиц, таких как em, rem, %, для размеров шрифта.
  • Пример:
    •   
    /*CSS*/

    html {
      font-size: 16px;
    }
    
    @media (max-width: 768px) {
      html {
        font-size: 14px; /* уменьшение размера шрифта для мобильных устройств */
      }
    }

Пример комплексной адаптивной верстки

  
    <!-- HTML -->

    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Адаптивная страница</title>
      <style>
        body {
          font-family: Arial, sans-serif;
          margin: 0;
          padding: 0;
        }
        .header {
          background-color: #333;
          color: white;
          text-align: center;
          padding: 10px;
        }
        .container {
          display: grid;
          grid-template-columns: 1fr;
          gap: 20px;
          padding: 20px;
        }
        .item {
          background-color: #f4f4f4;
          padding: 20px;
          border: 1px solid #ddd;
        }
        @media (min-width: 768px) {
          .container {
            grid-template-columns: repeat(2, 1fr);
          }
        }
        @media (min-width: 1200px) {
          .container {
            grid-template-columns: repeat(4, 1fr);
          }
        }
      </style>
    </head>
    <body>
      <div class="header">Моя Адаптивная Страница</div>
      <div class="container">
        <div class="item">Контент 1</div>
        <div class="item">Контент 2</div>
        <div class="item">Контент 3</div>
        <div class="item">Контент 4</div>
      </div>
    </body>
    </html>

Заключение

Адаптивная верстка играет важную роль в современном веб-дизайне, обеспечивая удобство использования на всех устройствах. Для создания таких макетов необходимо хорошо понимать работу с медиа-запросами, гибкими сетками и изображениями, а также подходами "мобильный в первую очередь" и "десктоп в первую очередь".
