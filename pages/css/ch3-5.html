Атрибутные селекторы.

Что такое атрибутные селекторы?

Атрибутные селекторы позволяют выбирать элементы на основе присутствия или значения определенного атрибута. Это полезно, если вы хотите стилизовать элементы с определенными атрибутами или их значениями, например, ссылки с атрибутом href, элементы input с разными типами, и многое другое.

Основной синтаксис

  
    /*CSS*/

    [атрибут] {
      /* стили */
    }

Виды атрибутных селекторов

1. Селектор по наличию атрибута

Используется для выбора всех элементов, у которых присутствует определенный атрибут, независимо от его значения.

  
    /*CSS*/

    [a] {
      color: green;
    }

Пример применения:

  
    <!-- HTML -->

    <a href="#">Ссылка с атрибутом href</a>
    <a name="section">Ссылка с атрибутом name</a>
    <div a="test">Див с атрибутом a</div>

Результат:

Ссылка с атрибутом href
Ссылка с атрибутом name
Див с атрибутом a

Все элементы с атрибутом a будут стилизованы.

2. Селектор по точному значению атрибута

Выбирает элементы, у которых значение атрибута точно совпадает с указанным.

  
    /*CSS*/

    [type="text"] {
      border: 1px solid red;
    }
  

  
    <!-- HTML -->

    <input type="text" /> <!-- Стилизуется -->
    <input type="password" /> <!-- Не стилизуется -->

Результат:

Элемент <input type="text"> получит заданные стили.

3. Селектор по частичному совпадению значений

  • Селектор с начальным совпадением (^=): выбирает элементы, значение атрибута которых начинается с определенной строки.
    •   
    /*CSS*/

    [href^="https"] {
      color: green;
    }

    Все ссылки, начинающиеся с https, станут зелеными.

  • Селектор с конечным совпадением ($=): выбирает элементы, значение атрибута которых заканчивается на определенную строку.
    •   
    /*CSS*/

    [href$=".pdf"] {
      color: red;
    }

    Все ссылки, ведущие на PDF-файлы, станут красными.

  • Селектор с подстрочным совпадением (*=): выбирает элементы, значение атрибута которых содержит определенную подстроку.
    •   
    /*CSS*/

    [title*="important"] {
      font-weight: bold;
    }

    Элементы с атрибутом title, содержащим слово "important", будут выделены жирным шрифтом.

Расширенные возможности атрибутных селекторов

4. Селектор с пробельным совпадением (~=)

Выбирает элементы, значение атрибута которых содержит указанное слово, разделенное пробелами.

  
    /*CSS*/

    [class~="highlight"] {
      background-color: yellow;
    }
  

  
    <!-- HTML -->

    <p class="text highlight important">Текст с выделением</p> <!-- Стилизуется -->
    <p class="highlighted-text">Другое слово не стилизуется</p> <!-- Не стилизуется -->

Результат:

Текст с выделением

Другое слово не стилизуется

Этот селектор подходит для выбора отдельных слов в строке, разделенной пробелами.

5. Селектор с совпадением по начальной строке в списке (|=)

Выбирает элементы, у которых значение атрибута начинается с указанной строки или совпадает с этой строкой. Чаще всего применяется для выбора элементов по языковому коду.

  
    /*CSS*/

    [lang|="en"] {
      font-style: italic;
    }
  

  
    <!-- HTML -->

    <p lang="en">Английский текст</p> <!-- Стилизуется -->
    <p lang="en-US">Американский английский текст</p> <!-- Стилизуется -->
    <p lang="fr">Французский текст</p> <!-- Не стилизуется -->

Результат:

Английский текст

Американский английский текст

Французский текст

Примеры практического применения

  1. Стилизация ссылок по типу ресурса:
    2.   
    /*CSS*/

    a[href^="https://"] {
      color: green;
    }
    a[href$=".pdf"] {
      color: red;
    }
    • Ссылки на безопасные сайты будут зелеными.
    • Ссылки на PDF-файлы будут красными.
  2. Стилизация элементов input по типу:
    3.   
    /*CSS*/

    input[type="text"] {
      border: 1px solid #ccc;
    }
    input[type="submit"] {
      background-color: blue;
      color: white;
    }

    Эти стили позволят различать визуально текстовые поля и кнопки отправки.

Важные аспекты атрибутных селекторов

  1. Совместимость с современными браузерами: большинство атрибутных селекторов поддерживаются во всех современных браузерах, но всегда стоит проверять совместимость для старых версий.
  2. Специфичность селекторов: атрибутные селекторы имеют такую же специфичность, как селекторы по тегу. Если у вас есть конфликтующие правила стилей, они могут быть переопределены более специфичными селекторами.
  3. Производительность: атрибутные селекторы могут быть менее производительными, особенно если применяются к большому количеству элементов на странице. Оптимизация селекторов может быть важна для производительности.

Атрибутные селекторы — это удобный и мощный инструмент, который помогает делать стили более гибкими и динамичными. Их использование позволяет эффективно управлять внешним видом элементов, основываясь на значениях атрибутов, без необходимости добавлять лишние классы или изменять HTML.
