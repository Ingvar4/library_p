Создание простого макета сайта с помощью Flexbox.

Давай разберем стилизацию веб-страницы с помощью CSS, с акцентом на Flexbox для создания макета. Flexbox (Flexible Box Layout) — это метод CSS, который упрощает создание гибких, адаптивных макетов. Flexbox позволяет располагать элементы вдоль одной оси (горизонтальной или вертикальной) и автоматически выравнивать их в зависимости от доступного пространства.

  1. Основные принципы Flexbox

    Flexbox состоит из родительского элемента, называемого "flex-контейнером", и дочерних элементов, называемых "flex-элементами". Flex-контейнер управляет расположением и поведением flex-элементов.

      
    /*CSS*/

    .container {
      display: flex;
    }
  2. Свойства flex-контейнера

    Эти свойства задаются контейнеру и влияют на поведение flex-элементов:

    • display: flex — задаёт контейнеру гибкий контейнер, активируя Flexbox.
    • flex-direction — определяет направление оси (строки или столбца):
      • row (по умолчанию): располагает элементы по горизонтали.
      • column: располагает элементы по вертикали.
      • row-reverse и column-reverse: изменяют порядок элементов в обратном направлении.
    • flex-wrap — управляет переносом flex-элементов:
      • nowrap (по умолчанию): элементы не переносятся на новую строку.
      • wrap: элементы переносятся на новую строку по мере необходимости.
      • wrap-reverse: переносится в обратном порядке.
    • justify-content — выравнивает flex-элементы по основной оси:
      • flex-start, center, flex-end, space-between, space-around, space-evenly.
    • align-items — выравнивает элементы по поперечной оси:
      • stretch (по умолчанию): элементы растягиваются, чтобы занять всю высоту контейнера.
      • flex-start, center, flex-end, baseline.
    • align-content — управляет выравниванием строк flex-элементов, если они переносятся.
  3. Свойства flex-элементов

    Эти свойства задаются каждому flex-элементу отдельно и влияют на его размер и поведение в контейнере:

    • flex-grow — определяет, насколько элемент может увеличиваться относительно других.
    • flex-shrink — определяет, насколько элемент может сжиматься при уменьшении пространства.
    • flex-basis — задает базовый размер элемента.
    • align-self — позволяет выровнять элемент по поперечной оси, независимо от других.
  4. Создание макета страницы с Flexbox

    5. Для практики создадим макет веб-страницы с заголовком, основным контентом и подвалом. Основной контент будет содержать боковую панель и область для основной статьи.

      
    <!-- HTML -->

    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Flexbox Layout</title>
      <link rel="stylesheet" href="styles.css">
    </head>
    <body>
      <header class="header">Заголовок</header>
      <div class="content">
        <aside class="sidebar">Боковая панель</aside>
        <main class="main">Основной контент</main>
      </div>
      <footer class="footer">Подвал</footer>
    </body>
    </html>
  

      
    /*CSS*/

    * {
      margin: 0;
      padding: 0;
      box-sizing: border-box;
    }
    
    body {
      font-family: Arial, sans-serif;
      display: flex;
      flex-direction: column;
      min-height: 100vh;
    }
    
    .header, .footer {
      background-color: #333;
      color: #fff;
      padding: 1rem;
      text-align: center;
    }
    
    .content {
      display: flex;
      flex: 1;
    }
    
    .sidebar {
      background-color: #f4f4f4;
      padding: 1rem;
      flex: 1;
    }
    
    .main {
      background-color: #e4e4e4;
      padding: 1rem;
      flex: 3;
    }

    Результат:

  5. Объяснение кода
    • header и footer — шапка и подвал с фиксированной высотой и централизованным текстом.
    • content — гибкий контейнер с помощью display: flex, внутри которого располагаются sidebar и main.
    • sidebar и main — задаем им соотношение flex (1:3), где основная область шире боковой панели.
  6. Использование адаптивности в Flexbox

    7. Flexbox позволяет создавать адаптивные макеты, которые изменяют своё расположение на разных устройствах.

      
    /*CSS*/

    @media (max-width: 768px) {
      .content {
        flex-direction: column;
      }
    }

Итог

С помощью Flexbox можно легко создать адаптивный макет с различным выравниванием и переносом элементов. Такой подход упрощает работу с расположением и структурой, особенно при изменении размеров экрана.
