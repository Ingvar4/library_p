HTML — гипертекстовый язык разметки, который мы используем для визуального и смыслового структурирования нашего web контента, например, определяем параграфы, заголовки, таблицы данных, или вставляем изображения и видео на страницу.

Этап 1: Введение в HTML

  1. Что такое HTML и как он работает?
    • Основные концепции веб-страниц: структура, браузеры и HTML.
    • Роль HTML в связке с CSS и JavaScript.
  2. Первая HTML-страница
    • Теги <html>, <head>, <body>, структура HTML-документа.
    • Тег <title>, заголовки <h1> – <h6>.
    • Практика: создание первой страницы с заголовком и текстом.

Этап 2: Текст и форматирование

  1. Параграфы и отступы
    • Теги <p>, <br>, <hr>.
    • Практика: создание текста с параграфами и разделительными линиями.
  2. Форматирование текста
    • Жирный текст <b>, курсив <i>, подчёркнутый <u>, зачёркнутый <s>.
    • Сильное выделение <strong> и акцентирование <em>.
    • Практика: форматирование абзацев с разными стилями текста.
  3. Списки
    • Нумерованные списки <ol> и ненумерованные <ul>, элементы списка <li>.
    • Практика: создание списков с различными типами пунктов.

Этап 3: Ссылки и изображения

  1. Гиперссылки
    • Тег <a>, атрибуты href, target.
    • Ссылки на внутренние страницы и внешние ресурсы.
    • Практика: создание набора гиперссылок.
  2. Изображения
    • Тег <img>, атрибуты src, alt, width, height.
    • Практика: вставка и настройка изображений на странице.

Этап 4: Структурирование содержимого

  1. Семантические теги
    • Основные структурные элементы: <header>, <footer>, <nav>, <main>, <article>, <section>, <aside>.
    • Важность семантики для SEO и доступности.
    • Практика: структурирование страницы с использованием семантических тегов.
  2. Таблицы
    • Теги <table>, <tr>, <td>, <th>, атрибуты colspan, rowspan.
    • Практика: создание таблиц с различными форматами.

Этап 5: Формы и взаимодействие с пользователем

  1. Формы
    • Основные теги: <form>, <input>, <label>, <textarea>, <button>.
    • Поля ввода: текстовые поля, пароли, чекбоксы, радиокнопки, селекторы.
    • Атрибуты: name, action, method, placeholder, required.
    • Практика: создание форм для сбора данных пользователей.
  2. Отправка данных
    • Понимание GET и POST методов.
    • Обработка отправки форм на сервер.

Этап 6: Мультимедиа и другие теги

  1. Аудио и видео
    • Теги <audio>, <video>, атрибуты controls, autoplay, loop.
    • Практика: вставка мультимедиа на веб-страницу.
  2. Интерактивные элементы
    • Теги <details>, <summary>, <dialog>, <progress>, <meter>.
    • Практика: работа с интерактивными элементами на странице.

Этап 7: Продвинутые темы

  1. Метаданные
    • Тег <meta>, атрибуты charset, name, content, настройка заголовков и описаний для SEO.
    • Практика: настройка метаданных для веб-страницы.
  2. Комментарии
    • Тег комментария <!-- комментарий -->.
    • Использование комментариев для пояснений в коде.
  3. Ссылки на внешние ресурсы
    • Подключение CSS и JavaScript: теги <link> и <script>.
    • Практика: подключение внешних файлов.

    • Основы адаптивности: использование семантических тегов и медиа-запросов.

Этап 9: Оптимизация и продвинутые техники

    • Работа с тегами для SEO оптимизации.
