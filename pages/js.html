JavaScript — язык программирования, который позволяет создавать динамически обновляемый контент, управлять мультимедиа, анимировать изображения и делать многое другое.

Часть 1: Введение в JavaScript и Основы

    День 1-2:

  • Что такое JavaScript и где его применяют
  • Установка среды разработки (VS Code, Node.js)
  • Подключение JavaScript к HTML-документу
  • Введение в синтаксис JavaScript (переменные, типы данных)

    День 3-4:

  • Операторы и выражения
  • Работа с переменными (let, const, var)
  • Базовые операции с числами и строками

    День 5-6:

  • Основы работы с функциями
  • Функции как блоки кода (определение, вызов)
  • Параметры и возврат значений

    День 7:

  • Практика: написание простых функций и выполнение арифметических операций

Часть 2: Управление потоком и основы массивов

    День 1-2:

  • Условные операторы (if, else, switch)
  • Логические операторы
  • Тернарный оператор

    День 3-4:

  • Циклы (for, while, do...while)
  • Вложенные циклы и управление потоком (break, continue)

    День 5-6:

  • Введение в массивы
  • Основные методы массивов (push, pop, shift, unshift)

    День 7:

  • Практика: создание простых программ с использованием циклов и массивов

Часть 3: Работа с объектами и обработка ошибок

    День 1-2:

  • Введение в объекты
  • Создание и доступ к свойствам объектов
  • Методы объектов

    День 3-4:

  • Введение в обработку ошибок
  • Конструкции try, catch, finally
  • Создание собственных ошибок (throw)

    День 5-6:

  • Глубокое копирование и сравнение объектов
  • Прототипы и наследование

    День 7:

  • Практика: создание и работа с объектами, реализация простых методов

Часть 4: Работа с DOM и событиями

    День 1-2:

  • Введение в Document Object Model (DOM)
  • Поиск и манипуляция элементами DOM (getElementById, querySelector)
  • Изменение стилей и содержимого элементов

    День 3-4:

  • Работа с событиями (click, input, submit)
  • Использование addEventListener для управления событиями

    День 5-6:

  • Создание и удаление элементов DOM
  • Работа с формами и валидация данных

    День 7:

  • Практика: создание интерактивной веб-страницы с обработкой событий

Часть 5: Асинхронное программирование и работа с API

    День 1-2:

  • Введение в асинхронное программирование
  • Использование setTimeout и setInterval
  • Введение в промисы (Promises)

    День 3-4:

  • Работа с async/await
  • Обработка ошибок в асинхронном коде

    День 5-6:

  • Введение в работу с API
  • Отправка запросов и получение данных (fetch, axios)

    День 7:

  • Практика: работа с API, получение данных и отображение их на странице

Часть 6: Завершение и проект

    День 1-2:

  • Повторение ключевых тем
  • Решение задач на алгоритмы и логику

    День 3-4:

  • Планирование проекта: создание ТЗ
  • Начало работы над финальным проектом (простое веб-приложение)

    День 5-6:

  • Продолжение работы над проектом
  • Реализация основных функций и взаимодействие с DOM и API

    День 7:

  • Завершение проекта и презентация (самостоятельная проверка и доработка)
